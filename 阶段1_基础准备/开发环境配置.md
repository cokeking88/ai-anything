# 开发环境配置（1周）

## 学习目标
搭建稳定、高效的AI开发环境，掌握必要的工具链。

## 环境配置原则
1. **隔离性**：使用虚拟环境避免包冲突
2. **可复现**：环境配置可复制、可迁移
3. **高效性**：选择适合的IDE和工具
4. **兼容性**：确保各组件兼容

## 第一天：Anaconda安装与配置

### 1.1 Anaconda安装
**Windows系统：**
1. 下载Anaconda安装包：https://www.anaconda.com/download
2. 运行安装程序，选择"Add Anaconda to my PATH environment variable"
3. 安装完成后重启命令行

**macOS系统：**
```bash
# 下载安装包
wget https://repo.anaconda.com/archive/Anaconda3-2024.02-1-MacOSX-x86_64.sh

# 运行安装
bash Anaconda3-2024.02-1-MacOSX-x86_64.sh

# 验证安装
conda --version
```

**Linux系统：**
```bash
# 下载安装包
wget https://repo.anaconda.com/archive/Anaconda3-2024.02-1-Linux-x86_64.sh

# 运行安装
bash Anaconda3-2024.02-1-Linux-x86_64.sh

# 初始化conda
conda init bash
source ~/.bashrc
```

### 1.2 验证安装
```bash
# 检查conda版本
conda --version

# 查看conda信息
conda info

# 更新conda
conda update conda
```

## 第二天：虚拟环境管理

### 2.1 创建AI学习环境
```bash
# 创建Python 3.9环境
conda create -n ai-learning python=3.9

# 激活环境
conda activate ai-learning

# 查看已创建环境
conda env list

# 删除环境
conda remove -n ai-learning --all
```

### 2.2 安装核心包
```bash
# 激活环境
conda activate ai-learning

# 安装科学计算包
conda install numpy pandas matplotlib seaborn scipy

# 安装机器学习包
conda install scikit-learn

# 安装Jupyter
conda install jupyter notebook

# 导出环境配置
conda env export > environment.yml

# 从配置文件创建环境
conda env create -f environment.yml
```

### 2.3 环境管理最佳实践
```bash
# 为不同项目创建不同环境
conda create -n project1 python=3.9
conda create -n project2 python=3.10

# 查看环境包列表
conda list

# 搜索包
conda search numpy

# 清理缓存
conda clean --all
```

## 第三天：IDE配置

### 3.1 VSCode配置
**安装扩展：**
1. Python
2. Pylance
3. Jupyter
4. GitLens
5. autoDocstring

**配置settings.json：**
```json
{
    "python.defaultInterpreterPath": "C:\\Users\\用户名\\anaconda3\\envs\\ai-learning\\python.exe",
    "python.terminal.activateEnvironment": true,
    "jupyter.notebookFileRoot": "${workspaceFolder}",
    "python.linting.enabled": true,
    "python.linting.pylintEnabled": true,
    "python.formatting.provider": "black",
    "editor.formatOnSave": true
}
```

### 3.2 PyCharm配置
**创建项目：**
1. File → New Project
2. 选择Pure Python
3. Interpreter type: Conda Environment
4. Existing environment: 选择ai-learning环境

**配置Python解释器：**
1. File → Settings → Project → Python Interpreter
2. 点击齿轮 → Add
3. Conda Environment → Existing environment
4. 选择路径：`anaconda3/envs/ai-learning/python.exe`

## 第六天：Docker基础

### 6.1 Docker安装
**Windows/macOS：**
下载Docker Desktop：https://www.docker.com/products/docker-desktop

**Linux：**
```bash
# Ubuntu
sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io

# 启动Docker
sudo systemctl start docker
sudo systemctl enable docker
```

### 6.2 基础命令
```bash
# 查看Docker版本
docker --version

# 拉取镜像
docker pull python:3.9

# 运行容器
docker run -it python:3.9 bash

# 查看运行中的容器
docker ps

# 查看所有容器
docker ps -a

# 停止容器
docker stop container_id

# 删除容器
docker rm container_id
```

### 6.3 AI开发Docker镜像
```dockerfile
# Dockerfile
FROM pytorch/pytorch:2.0.0-cuda11.7-cudnn8-runtime

WORKDIR /app

# 安装常用包
RUN pip install numpy pandas matplotlib seaborn scikit-learn jupyter

# 暴露Jupyter端口
EXPOSE 8888

# 启动Jupyter
CMD ["jupyter", "lab", "--ip=0.0.0.0", "--port=8888", "--no-browser", "--allow-root"]
```

## 常见问题解决

### 环境问题
1. **包冲突**：使用虚拟环境隔离
2. **安装失败**：检查Python版本兼容性
3. **导入错误**：确认包已安装到正确环境

### IDE问题
1. **解释器错误**：检查IDE中的Python解释器路径
2. **代码补全失效**：重启IDE或重新加载项目
3. **Jupyter内核问题**：重新安装ipykernel

### 性能问题
1. **内存不足**：使用更小的数据集或批处理
2. **计算速度慢**：考虑使用GPU或优化代码
