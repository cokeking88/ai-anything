## 第一周：线性代数

### 第1-2天：向量与矩阵基础
**学习内容：**

#### 1. 向量基础
- **定义**：有大小和方向的量，表示为 v = [v₁, v₂, v₃]（3维向量）
- **几何意义**：空间中的箭头，从原点指向点(v₁, v₂, v₃)
- **基本运算**：
  - 加法：对应元素相加
  - 标量乘法：每个元素乘以标量
- **点积（内积）**：
  - 定义：v1·v2 = v₁₁×v₂₁ + v₁₂×v₂₂ + v₁₃×v₂₃
  - 结果：标量（数值）
  - **几何意义**：
    1. **投影关系**：v1·v2 = |v1| × |v2| × cos(θ)，θ为两向量夹角
       - 点积 = v1的长度 × v2在v1方向上的投影长度
       - 点积 = v2的长度 × v1在v2方向上的投影长度
    2. **夹角判断**：
       - 点积 > 0：两向量夹角 < 90°（方向相近）
       - 点积 = 0：两向量夹角 = 90°（正交垂直）
       - 点积 < 0：两向量夹角 > 90°（方向相反）
    3. **相似度度量**：
       - 归一化后：cos(θ) = (v1·v2) / (|v1| × |v2|)
       - 值域：[-1, 1]
       - 1表示完全相同方向，0表示垂直，-1表示完全相反
  - **AI应用详解**：
    - **相似度计算**：
      - 在推荐系统中：用户特征向量与物品特征向量的点积表示用户对物品的偏好程度
      - 在NLP中：词向量（Word2Vec）的点积表示词语的语义相似度
      - 在图像识别中：图像特征向量的点积表示图像的相似程度
    - **特征加权**：
      - 在神经网络中：权重向量与输入向量的点积 = 加权求和
      - 公式：y = w·x + b = Σ(wᵢ × xᵢ) + b
      - 每个输入特征xᵢ通过权重wᵢ进行加权，点积实现特征的重要性加权
    - **注意力机制**：
      - 在Transformer中：Query与Key的点积计算注意力分数
      - 公式：Attention(Q,K,V) = softmax(Q·Kᵀ / √dₖ) × V
      - 点积衡量Query和Key的相关性，决定信息的重要性权重
- **叉积（外积）**：
  - **维度适用性**：
    - 内积（点积）：适用于任意维度（2D、3D、nD都可）
    - 叉积（外积）：**严格仅适用于3D向量**（2D向量需要扩展到3D计算）
  - **计算公式**：
    - v1 = [a₁, a₂, a₃]，v2 = [b₁, b₂, b₃]
    - v1 × v2 = [a₂b₃ - a₃b₂, a₃b₁ - a₁b₃, a₁b₂ - a₂b₁]
    - 记忆口诀：x分量去掉x行x列，y分量取负，z分量去掉z行z列
    - 行列式形式：
      ```
      | i   j   k  |
      | a₁  a₂  a₃ | = i(a₂b₃-a₃b₂) - j(a₁b₃-a₃b₁) + k(a₁b₂-a₂b₁)
      | b₁  b₂  b₃ |
      ```
  - **结果特点**：
    - 结果是新向量，垂直于原两个向量所在的平面
    - 长度 = |v1| × |v2| × sin(θ)（等于两向量构成的平行四边形面积）
    - 方向由右手定则确定
  - **AI应用详解**：
    - **计算机图形学**：
      - 计算平面法向量：用于光照计算、阴影渲染
      - 判断多边形朝向：正面/背面剔除
      - 计算旋转轴：两个方向向量的叉积得到旋转轴
    - **机器人学**：
      - 计算关节旋转方向
      - 路径规划中的方向判断
    - **物理引擎**：
      - 计算力矩：力臂 × 力 = 力矩向量
      - 角动量计算

#### 2. 矩阵基础
- **定义**：二维数字阵列，m×n矩阵有m行n列
- **基本运算**：
  - 加法：对应位置元素相加（需同维度）
  - 标量乘法：每个元素乘以标量

#### 3. 矩阵乘法详解
- **定义**：A(m×n) × B(n×p) = C(m×p)
  - 规则：A的列数必须等于B的行数
  - 计算：C[i][j] = Σ A[i][k] × B[k][j]（A的第i行与B的第j列点积）
- **数学意义**：
  1. **线性变换的复合**：
     - 矩阵表示线性变换（旋转、缩放、剪切）
     - 矩阵乘法 = 两个变换的叠加
     - 例：先旋转再缩放 = 旋转矩阵 × 缩放矩阵
  2. **基底变换**：
     - 矩阵的列向量 = 新坐标系的基底
     - 矩阵乘法 = 将向量从一个坐标系变换到另一个坐标系
  3. **系统方程组的解**：
     - Ax = b：A是系数矩阵，x是未知向量，b是结果向量
     - 矩阵乘法 = 方程组的向量化表示
- **AI应用详解**：
  - **神经网络层间计算**：
    - y = W×x + b
    - W：权重矩阵（学习参数）
    - x：输入特征向量
    - b：偏置向量
    - 点积实现特征的加权组合
  - **特征变换**：
    - 降维：PCA中用特征向量矩阵将数据投影到主成分空间
    - 升维：将数据映射到更高维空间（核方法）
  - **注意力机制**：
    - Q×Kᵀ：计算查询与键的相关性
    - softmax(Q×Kᵀ/√d)×V：加权聚合值向量

#### 4. 矩阵转置详解
- **定义**：行列互换，(Aᵀ)[i][j] = A[j][i]
  - m×n矩阵转置后变为n×m矩阵
  - 第i行变为第i列，第j列变为第j行
- **数学意义**：
  1. **行向量与列向量的转换**：
     - 行向量 [1, 2, 3] 转置后变为列向量 [1, 2, 3]ᵀ
     - 在数学表示中，向量默认为列向量
  2. **内积的矩阵形式**：
     - a·b = aᵀ×b（行向量×列向量=标量）
     - 转置使向量可以相乘得到标量
  3. **对称性**：
     - 对称矩阵：A = Aᵀ
     - 协方差矩阵、相关矩阵都是对称矩阵
  4. **正交变换**：
     - 正交矩阵：Aᵀ×A = I（转置=逆矩阵）
     - 旋转矩阵、反射矩阵都是正交矩阵
- **AI应用详解**：
  - **数据格式转换**：
    - 将行存储的数据转为列存储，便于批量计算
    - 特征矩阵X：每行是一个样本，转置后每列是一个样本
  - **计算相似度矩阵**：
    - X×Xᵀ：计算所有样本两两之间的相似度
    - 结果是m×m矩阵，(i,j)位置表示第i个和第j个样本的相似度
  - **梯度计算**：
    - 反向传播中，权重的梯度 = 输入ᵀ × 输出梯度
    - 转置用于维度匹配

#### 5. 特殊矩阵
- **单位矩阵**：主对角线为1，其余为0
  - 性质：A×I = I×A = A（乘法恒等元）
- **对角矩阵**：仅主对角线有非零元素
  - 性质：对角矩阵×向量 = 向量各分量分别乘以对角元素
- **零矩阵**：所有元素为0
  - 性质：A×0 = 0×A = 0

#### 6. 特征值与特征向量详解
- **定义**：
  - 特征方程：A×v = λ×v
    - A：方阵（n×n）
    - v：特征向量（非零向量）
    - λ：特征值（标量）
  - 含义：矩阵A对向量v的作用 = 简单地将v拉伸λ倍（方向不变或反向）

- **数学意义**：
  1. **矩阵的"本征方向"**：
     - 特征向量 = 矩阵变换后方向不变的向量
     - 特征值 = 该方向上的拉伸/压缩比例
     - 例：旋转矩阵的特征向量是旋转轴方向

  2. **矩阵的对角化**：
     - 特征分解：A = P×D×P⁻¹
       - P：特征向量组成的矩阵
       - D：特征值组成的对角矩阵
     - 对角化 = 将复杂变换分解为简单的缩放变换

  3. **稳定性分析**：
     - 特征值的大小反映系统稳定性
     - |λ| < 1：该方向衰减
     - |λ| > 1：该方向发散
     - |λ| = 1：该方向保持

  4. **二次型分析**：
     - 对称矩阵的特征值 > 0：正定矩阵
     - 对称矩阵的特征值 < 0：负定矩阵
     - 特征值符号决定二次型的凹凸性

- **AI应用详解**：
  1. **PCA降维（主成分分析）**：
     - 原理：协方差矩阵的特征向量 = 数据变化的主要方向
     - 特征值大小 = 该方向上的方差大小
     - 应用：保留大特征值对应的特征向量，丢弃小特征值方向
     - 效果：用少数主成分表示大部分数据信息

  2. **推荐系统**：
     - 用户-物品评分矩阵的特征分解
     - 特征向量 = 用户/物品的潜在因子
     - 特征值 = 该因子的重要性
     - 应用：矩阵补全、协同过滤

  3. **图像压缩**：
     - 图像矩阵的特征分解
     - 保留主要特征值对应的特征向量
     - 用低秩近似表示图像
     - 效果：大幅减少存储空间

  4. **PageRank算法**：
     - 网页链接矩阵的特征向量
     - 主特征向量 = 网页重要性排名
     - Google搜索引擎的核心算法

  5. **谱聚类**：
     - 相似度矩阵的特征分解
     - 特征向量 = 数据的低维嵌入
     - 在低维空间进行聚类

#### 7. 矩阵乘法在神经网络中
- 层间计算：y = W×x + b
- W：权重矩阵，x：输入向量，b：偏置向量，y：输出向量

**AI应用理解：**
- **特征向量**：数据表示、数据的主要变化方向
- **矩阵乘法**：神经网络层间计算、特征变换
- **转置**：数据变换、维度匹配
- **特征分解**：PCA降维、推荐系统、数据压缩

```python
import numpy as np

# 特征值分解
A = np.array([[4, 2], [1, 3]])
eigenvalues, eigenvectors = np.linalg.eig(A)

print(f"特征值：{eigenvalues}")
```

### 第5-7天：矩阵运算进阶
**学习内容：**

#### 1. 逆矩阵详解
- **定义**：
  - A×A⁻¹ = A⁻¹×A = I（单位矩阵）
  - 逆矩阵存在条件：矩阵必须是方阵且行列式不为0
  - 可逆矩阵 = 非奇异矩阵 = 满秩矩阵

- **数学意义**：
  1. **变换的逆操作**：
     - 矩阵表示变换，逆矩阵表示"撤销"该变换
     - 例：旋转矩阵的逆矩阵 = 反向旋转矩阵
     - 例：缩放矩阵的逆矩阵 = 反向缩放矩阵

  2. **方程组求解**：
     - Ax = b → x = A⁻¹×b
     - 逆矩阵直接给出方程组的解

  3. **坐标变换**：
     - 从旧坐标系到新坐标系：x_new = T×x_old
     - 从新坐标系回到旧坐标系：x_old = T⁻¹×x_new

- **AI应用**：
  - **线性回归的解析解**：
    - 最小二乘法：θ = (Xᵀ×X)⁻¹×Xᵀ×y
    - 需要计算(Xᵀ×X)的逆矩阵
  - **正则化问题**：
    - 当Xᵀ×X不可逆时，添加正则项：θ = (Xᵀ×X + λI)⁻¹×Xᵀ×y
  - **数据标准化**：
    - Z-score标准化的逆变换：x = z×σ + μ

#### 2. 行列式详解
- **定义**：
  - det(A)或|A|：方阵的标量值
  - 2×2矩阵：det([[a,b],[c,d]]) = a×d - b×c
  - n×n矩阵：按行/列展开计算

- **数学意义**：
  1. **体积缩放因子**：
     - |det(A)| = 矩阵A变换后的体积缩放比例
     - det(A) = 2：面积/体积扩大2倍
     - det(A) = 0.5：面积/体积缩小到0.5倍

  2. **方向判断**：
     - det(A) > 0：保持方向（旋转、缩放）
     - det(A) < 0：反转方向（包含反射）
     - det(A) = 0：降维变换（压缩到低维空间）

  3. **可逆性判断**：
     - det(A) ≠ 0：矩阵可逆
     - det(A) = 0：矩阵不可逆（奇异矩阵）

  4. **特征值关系**：
     - det(A) = 所有特征值的乘积
     - det(A) = λ₁×λ₂×...×λₙ

- **AI应用**：
  - **数据预处理**：
    - 检查数据矩阵是否满秩
    - 判断特征是否线性相关
  - **概率密度计算**：
    - 多元正态分布：f(x) ∝ 1/|Σ|^(1/2)
    - 协方差矩阵的行列式影响概率密度
  - **雅可比行列式**：
    - 变量变换中的概率密度调整
    - 生成模型中的密度估计

#### 3. 矩阵的秩详解
- **定义**：
  - 矩阵的秩 = 线性无关的行/列向量的最大个数
  - 秩 ≤ min(行数, 列数)
  - 满秩：秩 = min(行数, 列数)

- **数学意义**：
  1. **线性独立性**：
     - 秩 = 矩阵中线性无关向量的个数
     - 秩 < 行数：存在线性相关的行
     - 秩 < 列数：存在线性相关的列

  2. **方程组解的情况**：
     - 秩 = 未知数个数：唯一解
     - 秩 < 未知数个数：无穷多解
     - 秩 > 未知数个数：无解（超定方程组）

  3. **矩阵分解**：
     - 秩决定了矩阵的"有效维度"
     - 低秩近似：用更少的信息表示矩阵

- **AI应用**：
  - **特征选择**：
    - 检查特征矩阵的秩
    - 秩 < 特征数：存在冗余特征
    - 移除线性相关的特征
  - **降维**：
    - 低秩矩阵可以用更少的维度表示
    - PCA、SVD都是基于秩的降维方法
  - **数据质量检查**：
    - 检查数据矩阵是否满秩
    - 不满秩可能表示数据缺失或异常

#### 4. 线性方程组求解详解
- **定义**：
  - Ax = b：A是系数矩阵，x是未知向量，b是常数向量
  - 求解方法：直接法、迭代法、最小二乘法

- **数学意义**：
  1. **直接求解**：
     - x = A⁻¹×b（当A可逆时）
     - 计算复杂度高，数值不稳定

  2. **高斯消元法**：
     - 将增广矩阵[A|b]化为行阶梯形
     - 回代求解
     - 计算复杂度O(n³)

  3. **LU分解**：
     - A = L×U（下三角×上三角）
     - 先解Ly = b，再解Ux = y
     - 适合多次求解不同b的情况

  4. **最小二乘法**：
     - 当方程组无解时（超定方程组）
     - 最小化||Ax - b||²
     - 解：x = (Aᵀ×A)⁻¹×Aᵀ×b

- **AI应用**：
  - **线性回归**：
    - 最小二乘法求解回归系数
    - θ = (Xᵀ×X)⁻¹×Xᵀ×y
  - **正则化回归**：
    - Ridge回归：θ = (Xᵀ×X + αI)⁻¹×Xᵀ×y
    - 解决多重共线性问题
  - **优化问题**：
    - 梯度下降的矩阵形式
    - 牛顿法中的Hessian矩阵求逆
  - **推荐系统**：
    - 矩阵补全问题
    - 求解低秩矩阵近似

**AI应用理解：**
- **最小二乘法**：线性回归的核心，求解最优参数
- **正则化**：避免矩阵不可逆，防止过拟合
- **数据变换**：可逆变换保持信息，不可逆变换降维
- **特征选择**：矩阵的秩反映特征的独立性

## 第二周：概率统计

### 第8-10天：概率基础
**学习内容：**

#### 1. 概率定义与基本规则
- **概率定义**：
  - 古典概率：P(A) = 有利结果数 / 所有可能结果数
  - 频率概率：P(A) = 事件A发生的频率（试验次数→∞）
  - 主观概率：基于经验和信念的概率估计

- **概率公理**：
  1. 非负性：P(A) ≥ 0
  2. 规范性：P(Ω) = 1（必然事件概率为1）
  3. 可加性：若A∩B=∅，则P(A∪B) = P(A) + P(B)

- **基本规则**：
  - **加法公式**：P(A∪B) = P(A) + P(B) - P(A∩B)
  - **补集公式**：P(A') = 1 - P(A)
  - **乘法公式**：P(A∩B) = P(A) × P(B|A)
  - **全概率公式**：P(B) = Σ P(Aᵢ) × P(B|Aᵢ)

- **AI应用**：
  - **朴素贝叶斯分类器**：基于概率的分类方法
  - **不确定性建模**：用概率表示模型的不确定性
  - **概率图模型**：用概率描述变量间的关系

#### 2. 条件概率详解
- **定义**：P(A|B) = P(A∩B) / P(B)
  - 含义：在事件B发生的条件下，事件A发生的概率
  - P(B) > 0 是条件概率的前提

- **理解要点**：
  - 条件概率改变了样本空间：从Ω缩小到B
  - P(A|B) ≠ P(A)：条件概率通常不等于无条件概率
  - P(A|B) + P(A'|B) = 1：条件下的互补事件

- **链式法则**：
  - P(A∩B∩C) = P(A) × P(B|A) × P(C|A∩B)
  - 推广：P(A₁∩A₂∩...∩Aₙ) = P(A₁) × P(A₂|A₁) × ... × P(Aₙ|A₁∩...∩Aₙ₋₁)

- **AI应用**：
  - **马尔可夫链**：P(Xₙ|X₁,...,Xₙ₋₁) = P(Xₙ|Xₙ₋₁)
  - **隐马尔可夫模型**：条件概率用于状态转移和观测生成
  - **贝叶斯网络**：用条件概率表描述变量依赖关系

#### 3. 贝叶斯定理详解
- **公式**：P(A|B) = P(B|A) × P(A) / P(B)
  - P(A|B)：后验概率（观察到B后对A的信念）
  - P(A)：先验概率（观察前对A的信念）
  - P(B|A)：似然（A成立时B出现的概率）
  - P(B)：证据（B出现的总概率）

- **展开形式**：P(A|B) = P(B|A) × P(A) / [P(B|A)×P(A) + P(B|A')×P(A')]

- **核心思想**：
  - 用新证据B更新对事件A的信念
  - 从先验概率得到后验概率
  - 结合先验知识和观测数据

- **AI应用详解**：
  - **朴素贝叶斯分类器**：
    - P(类别|特征) = P(特征|类别) × P(类别) / P(特征)
    - 假设特征之间条件独立
    - 应用：垃圾邮件分类、情感分析

  - **贝叶斯优化**：
    - 用高斯过程建模目标函数
    - 用贝叶斯定理更新后验分布
    - 选择下一个采样点

  - **贝叶斯神经网络**：
    - 权重不是固定值，而是分布
    - 用贝叶斯定理更新权重的后验分布
    - 提供模型的不确定性估计

  - **AB测试**：
    - 用贝叶斯方法判断哪个版本更好
    - 实时更新概率估计
    - 提前终止不好的实验

#### 4. 独立性详解
- **定义**：
  - 事件独立：P(A∩B) = P(A) × P(B)
  - 等价条件：P(A|B) = P(A) 或 P(B|A) = P(B)
  - 含义：一个事件的发生不影响另一个事件的概率

- **条件独立**：
  - 定义：P(A∩B|C) = P(A|C) × P(B|C)
  - 含义：在C条件下，A和B独立
  - 朴素贝叶斯的关键假设：特征之间条件独立

- **独立vs不相关**：
  - 独立 ⇒ 不相关（协方差为0）
  - 不相关 ⇏ 独立（可能有非线性关系）

- **AI应用**：
  - **朴素贝叶斯**：假设特征之间条件独立
  - **集成学习**：基学习器之间的独立性很重要
  - **随机森林**：通过随机特征选择增加独立性

**AI应用理解：**
- **朴素贝叶斯分类器**：基于贝叶斯定理和条件独立假设的分类方法
- **概率图模型**：用概率描述变量间的依赖关系
- **贝叶斯优化**：用贝叶斯方法进行超参数优化
- **不确定性估计**：用概率表示模型预测的置信度

### 第11-14天：统计分布
**学习内容：**

#### 1. 正态分布（高斯分布）
- **定义**：X ~ N(μ, σ²)
  - 概率密度函数：f(x) = (1/√(2πσ²)) × e^(-(x-μ)²/(2σ²))
  - μ：均值（分布中心位置）
  - σ²：方差（分布离散程度）
  - σ：标准差

- **数学意义**：
  1. **对称性**：关于均值μ对称，左右形状相同
  2. **68-95-99.7规则**：
     - 68%数据在[μ-σ, μ+σ]范围内
     - 95%数据在[μ-2σ, μ+2σ]范围内
     - 99.7%数据在[μ-3σ, μ+3σ]范围内
  3. **钟形曲线**：中间高，两边低，无限延伸
  4. **标准化**：Z = (X-μ)/σ ~ N(0,1)（标准正态分布）

- **AI应用**：
  - **数据标准化**：假设数据服从正态分布，进行Z-score标准化
  - **误差建模**：许多测量误差服从正态分布
  - **高斯过程**：用于贝叶斯优化和回归
  - **生成模型**：变分自编码器(VAE)的隐变量假设
  - **权重初始化**：神经网络权重常用正态分布初始化

#### 2. 伯努利分布
- **定义**：X ~ Bernoulli(p)
  - 只有两种结果：成功(1)和失败(0)
  - P(X=1) = p, P(X=0) = 1-p
  - 期望：E[X] = p
  - 方差：Var(X) = p(1-p)

- **数学意义**：
  1. **最简单的离散分布**：单次试验的成功/失败
  2. **参数p**：成功的概率，取值[0,1]
  3. **方差最大**：当p=0.5时，方差最大为0.25

- **AI应用**：
  - **二分类问题**：输出层用Sigmoid激活，预测伯努利分布
  - **逻辑回归**：假设标签服从伯努利分布
  - **Dropout**：伯努利分布决定神经元是否保留
  - **A/B测试**：用户是否点击服从伯努利分布

#### 3. 二项分布
- **定义**：X ~ Binomial(n, p)
  - n次独立伯努利试验中成功的次数
  - P(X=k) = C(n,k) × p^k × (1-p)^(n-k)
  - 期望：E[X] = np
  - 方差：Var(X) = np(1-p)

- **数学意义**：
  1. **多次试验**：n次独立伯努利试验的和
  2. **离散分布**：k取值0,1,2,...,n
  3. **正态近似**：当n很大时，近似正态分布N(np, np(1-p))

- **AI应用**：
  - **分类准确率**：n个样本中正确分类的个数
  - **多标签分类**：每个标签是伯努利分布，整体是二项分布
  - **假设检验**：A/B测试中的显著性检验
  - **强化学习**：多次尝试中的成功次数

#### 4. 泊松分布
- **定义**：X ~ Poisson(λ)
  - 单位时间/空间内事件发生次数
  - P(X=k) = (λ^k × e^(-λ)) / k!
  - λ：单位时间/空间内事件的平均发生次数
  - 期望：E[X] = λ
  - 方差：Var(X) = λ

- **数学意义**：
  1. **稀有事件**：描述稀有事件在固定时间/空间内的发生次数
  2. **无记忆性**：事件发生是独立的
  3. **λ=均值=方差**：泊松分布的独特性质

- **AI应用**：
  - **异常检测**：单位时间内异常事件的次数
  - **流量预测**：网站访问量、服务器请求量
  - **自然语言处理**：文档中某词出现的次数
  - **排队论**：客服呼叫中心的来电数量

#### 5. 指数分布
- **定义**：X ~ Exponential(λ)
  - 两次连续事件发生之间的时间间隔
  - f(x) = λ × e^(-λx), x ≥ 0
  - λ：事件发生率
  - 期望：E[X] = 1/λ
  - 方差：Var(X) = 1/λ²

- **数学意义**：
  1. **连续分布**：描述时间间隔
  2. **无记忆性**：P(X>s+t|X>s) = P(X>t)
  3. **与泊松分布的关系**：泊松过程中事件间隔服从指数分布

- **AI应用**：
  - **生存分析**：设备故障时间、用户流失时间
  - **排队论**：服务时间、等待时间
  - **可靠性工程**：产品寿命分析
  - **推荐系统**：用户下次访问的时间间隔

#### 6. 均匀分布
- **定义**：X ~ Uniform(a, b)
  - 在[a,b]区间内等概率取值
  - f(x) = 1/(b-a), a ≤ x ≤ b
  - 期望：E[X] = (a+b)/2
  - 方差：Var(X) = (b-a)²/12

- **数学意义**：
  1. **等概率**：区间内任意点概率密度相同
  2. **最简单连续分布**：常用于随机数生成
  3. **最大熵分布**：在给定区间内，均匀分布的熵最大

- **AI应用**：
  - **随机初始化**：神经网络权重的均匀分布初始化
  - **超参数搜索**：网格搜索、随机搜索
  - **数据增强**：在一定范围内随机扰动数据
  - **蒙特卡洛方法**：生成随机样本

#### 7. 期望与方差
- **期望（均值）**：
  - 离散：E[X] = Σ xᵢ × P(xᵢ)
  - 连续：E[X] = ∫ x × f(x) dx
  - 含义：随机变量的平均值，分布的中心位置

- **方差**：
  - 公式：Var(X) = E[(X-μ)²] = E[X²] - (E[X])²
  - 含义：随机变量取值的离散程度
  - 标准差：σ = √Var(X)

- **AI应用**：
  - **损失函数**：MSE损失 = 预测值与真实值的方差
  - **正则化**：L2正则化限制权重的方差
  - **批量归一化**：调整每层输出的均值和方差

#### 8. 大数定律
- **定义**：当试验次数n→∞时，样本均值收敛于总体均值
  - 弱大数定律：依概率收敛
  - 强大数定律：几乎必然收敛

```python
# 数值导数
def numerical_derivative(f, x, h=1e-5):
    """计算函数在x处的数值导数"""
    return (f(x + h) - f(x - h)) / (2 * h)

# 测试函数
def f(x):
    return x**2 + 2*x + 1

# 计算导数
x = 2
derivative = numerical_derivative(f, x)
print(f"函数 f(x)=x²+2x+1 在 x={x} 处的导数：{derivative:.4f}")
print(f"理论导数：2*x+2 = {2*x+2}")

# 多元函数梯度
def gradient_2d(f, x, y, h=1e-5):
    """计算二元函数的梯度"""
    df_dx = (f(x + h, y) - f(x - h, y)) / (2 * h)
    df_dy = (f(x, y + h) - f(x, y - h)) / (2 * h)
    return np.array([df_dx, df_dy])

# 测试函数：f(x,y) = x² + y²
def f_2d(x, y):
    return x**2 + y**2

# 计算梯度
x, y = 1, 2
grad = gradient_2d(f_2d, x, y)
print(f"函数 f(x,y)=x²+y² 在点({x},{y})处的梯度：{grad}")
print(f"理论梯度：(2x, 2y) = ({2*x}, {2*y})")
```

```python
import numpy as np
import matplotlib.pyplot as plt

# 梯度下降法实现
def gradient_descent(f, gradient, start, learning_rate=0.1, n_iterations=100):
    """梯度下降算法"""
    x = start
    history = [x]
    
    for i in range(n_iterations):
        grad = gradient(x)
        x = x - learning_rate * grad
        history.append(x)
    
    return x, history

# 测试函数：f(x) = x²

    return x**2

def gradient_f(x):
    return 2*x
```

```python
import numpy as np

# 熵计算
def entropy(probabilities):
    """计算熵 H(p) = -∑ p(x) * log2(p(x))"""
    # 移除概率为0的项
    p = probabilities[probabilities > 0]
    return -np.sum(p * np.log2(p))

# 示例：抛硬币
fair_coin = np.array([0.5, 0.5])  # 公平硬币
biased_coin = np.array([0.9, 0.1])  # 偏硬币

H_fair = entropy(fair_coin)
H_biased = entropy(biased_coin)

print(f"公平硬币的熵：{H_fair:.4f}")
print(f"偏硬币的熵：{H_biased:.4f}")

# KL散度
def kl_divergence(p, q):
    """计算KL散度 D_KL(p||q) = ∑ p(x) * log(p(x)/q(x))"""
    # 移除概率为0的项
    mask = (p > 0) & (q > 0)
    p = p[mask]
    q = q[mask]
    return np.sum(p * np.log(p / q))

# 示例
p = np.array([0.5, 0.5])
q = np.array([0.9, 0.1])

kl = kl_divergence(p, q)
print(f"KL散度 D_KL(p||q)：{kl:.4f}")

# 交叉熵
def cross_entropy(p, q):
    """计算交叉熵 H(p,q) = -∑ p(x) * log(q(x))"""
    # 移除概率为0的项
    mask = (p > 0) & (q > 0)
    p = p[mask]
    q = q[mask]
    return -np.sum(p * np.log(q))

ce = cross_entropy(p, q)
print(f"交叉熵 H(p,q)：{ce:.4f}")
```
