## 第一周：线性代数

### 第1-2天：向量与矩阵基础
**学习内容：**

#### 1. 向量基础
- **定义**：有大小和方向的量，表示为 v = [v₁, v₂, v₃]（3维向量）
- **几何意义**：空间中的箭头，从原点指向点(v₁, v₂, v₃)
- **基本运算**：
  - 加法：对应元素相加
  - 标量乘法：每个元素乘以标量
- **点积（内积）**：
  - 定义：v1·v2 = v₁₁×v₂₁ + v₁₂×v₂₂ + v₁₃×v₂₃
  - 结果：标量（数值）
  - **几何意义**：
    1. **投影关系**：v1·v2 = |v1| × |v2| × cos(θ)，θ为两向量夹角
       - 点积 = v1的长度 × v2在v1方向上的投影长度
       - 点积 = v2的长度 × v1在v2方向上的投影长度
    2. **夹角判断**：
       - 点积 > 0：两向量夹角 < 90°（方向相近）
       - 点积 = 0：两向量夹角 = 90°（正交垂直）
       - 点积 < 0：两向量夹角 > 90°（方向相反）
    3. **相似度度量**：
       - 归一化后：cos(θ) = (v1·v2) / (|v1| × |v2|)
       - 值域：[-1, 1]
       - 1表示完全相同方向，0表示垂直，-1表示完全相反
  - **AI应用详解**：
    - **相似度计算**：
      - 在推荐系统中：用户特征向量与物品特征向量的点积表示用户对物品的偏好程度
      - 在NLP中：词向量（Word2Vec）的点积表示词语的语义相似度
      - 在图像识别中：图像特征向量的点积表示图像的相似程度
    - **特征加权**：
      - 在神经网络中：权重向量与输入向量的点积 = 加权求和
      - 公式：y = w·x + b = Σ(wᵢ × xᵢ) + b
      - 每个输入特征xᵢ通过权重wᵢ进行加权，点积实现特征的重要性加权
    - **注意力机制**：
      - 在Transformer中：Query与Key的点积计算注意力分数
      - 公式：Attention(Q,K,V) = softmax(Q·Kᵀ / √dₖ) × V
      - 点积衡量Query和Key的相关性，决定信息的重要性权重
- **叉积（外积）**：
  - **维度适用性**：
    - 内积（点积）：适用于任意维度（2D、3D、nD都可）
    - 叉积（外积）：**严格仅适用于3D向量**（2D向量需要扩展到3D计算）
  - **计算公式**：
    - v1 = [a₁, a₂, a₃]，v2 = [b₁, b₂, b₃]
    - v1 × v2 = [a₂b₃ - a₃b₂, a₃b₁ - a₁b₃, a₁b₂ - a₂b₁]
    - 记忆口诀：x分量去掉x行x列，y分量取负，z分量去掉z行z列
    - 行列式形式：
      ```
      | i   j   k  |
      | a₁  a₂  a₃ | = i(a₂b₃-a₃b₂) - j(a₁b₃-a₃b₁) + k(a₁b₂-a₂b₁)
      | b₁  b₂  b₃ |
      ```
  - **结果特点**：
    - 结果是新向量，垂直于原两个向量所在的平面
    - 长度 = |v1| × |v2| × sin(θ)（等于两向量构成的平行四边形面积）
    - 方向由右手定则确定
  - **AI应用详解**：
    - **计算机图形学**：
      - 计算平面法向量：用于光照计算、阴影渲染
      - 判断多边形朝向：正面/背面剔除
      - 计算旋转轴：两个方向向量的叉积得到旋转轴
    - **机器人学**：
      - 计算关节旋转方向
      - 路径规划中的方向判断
    - **物理引擎**：
      - 计算力矩：力臂 × 力 = 力矩向量
      - 角动量计算

#### 2. 矩阵基础
- **定义**：二维数字阵列，m×n矩阵有m行n列
- **基本运算**：
  - 加法：对应位置元素相加（需同维度）
  - 标量乘法：每个元素乘以标量

#### 3. 矩阵乘法详解
- **定义**：A(m×n) × B(n×p) = C(m×p)
  - 规则：A的列数必须等于B的行数
  - 计算：C[i][j] = Σ A[i][k] × B[k][j]（A的第i行与B的第j列点积）
- **数学意义**：
  1. **线性变换的复合**：
     - 矩阵表示线性变换（旋转、缩放、剪切）
     - 矩阵乘法 = 两个变换的叠加
     - 例：先旋转再缩放 = 旋转矩阵 × 缩放矩阵
  2. **基底变换**：
     - 矩阵的列向量 = 新坐标系的基底
     - 矩阵乘法 = 将向量从一个坐标系变换到另一个坐标系
  3. **系统方程组的解**：
     - Ax = b：A是系数矩阵，x是未知向量，b是结果向量
     - 矩阵乘法 = 方程组的向量化表示
- **AI应用详解**：
  - **神经网络层间计算**：
    - y = W×x + b
    - W：权重矩阵（学习参数）
    - x：输入特征向量
    - b：偏置向量
    - 点积实现特征的加权组合
  - **特征变换**：
    - 降维：PCA中用特征向量矩阵将数据投影到主成分空间
    - 升维：将数据映射到更高维空间（核方法）
  - **注意力机制**：
    - Q×Kᵀ：计算查询与键的相关性
    - softmax(Q×Kᵀ/√d)×V：加权聚合值向量

#### 4. 矩阵转置详解
- **定义**：行列互换，(Aᵀ)[i][j] = A[j][i]
  - m×n矩阵转置后变为n×m矩阵
  - 第i行变为第i列，第j列变为第j行
- **数学意义**：
  1. **行向量与列向量的转换**：
     - 行向量 [1, 2, 3] 转置后变为列向量 [1, 2, 3]ᵀ
     - 在数学表示中，向量默认为列向量
  2. **内积的矩阵形式**：
     - a·b = aᵀ×b（行向量×列向量=标量）
     - 转置使向量可以相乘得到标量
  3. **对称性**：
     - 对称矩阵：A = Aᵀ
     - 协方差矩阵、相关矩阵都是对称矩阵
  4. **正交变换**：
     - 正交矩阵：Aᵀ×A = I（转置=逆矩阵）
     - 旋转矩阵、反射矩阵都是正交矩阵
- **AI应用详解**：
  - **数据格式转换**：
    - 将行存储的数据转为列存储，便于批量计算
    - 特征矩阵X：每行是一个样本，转置后每列是一个样本
  - **计算相似度矩阵**：
    - X×Xᵀ：计算所有样本两两之间的相似度
    - 结果是m×m矩阵，(i,j)位置表示第i个和第j个样本的相似度
  - **梯度计算**：
    - 反向传播中，权重的梯度 = 输入ᵀ × 输出梯度
    - 转置用于维度匹配

#### 5. 特殊矩阵
- **单位矩阵**：主对角线为1，其余为0
  - 性质：A×I = I×A = A（乘法恒等元）
- **对角矩阵**：仅主对角线有非零元素
  - 性质：对角矩阵×向量 = 向量各分量分别乘以对角元素
- **零矩阵**：所有元素为0
  - 性质：A×0 = 0×A = 0

#### 6. 特征值与特征向量详解
- **定义**：
  - 特征方程：A×v = λ×v
    - A：方阵（n×n）
    - v：特征向量（非零向量）
    - λ：特征值（标量）
  - 含义：矩阵A对向量v的作用 = 简单地将v拉伸λ倍（方向不变或反向）

- **数学意义**：
  1. **矩阵的"本征方向"**：
     - 特征向量 = 矩阵变换后方向不变的向量
     - 特征值 = 该方向上的拉伸/压缩比例
     - 例：旋转矩阵的特征向量是旋转轴方向

  2. **矩阵的对角化**：
     - 特征分解：A = P×D×P⁻¹
       - P：特征向量组成的矩阵
       - D：特征值组成的对角矩阵
     - 对角化 = 将复杂变换分解为简单的缩放变换

  3. **稳定性分析**：
     - 特征值的大小反映系统稳定性
     - |λ| < 1：该方向衰减
     - |λ| > 1：该方向发散
     - |λ| = 1：该方向保持

  4. **二次型分析**：
     - 对称矩阵的特征值 > 0：正定矩阵
     - 对称矩阵的特征值 < 0：负定矩阵
     - 特征值符号决定二次型的凹凸性

- **AI应用详解**：
  1. **PCA降维（主成分分析）**：
     - 原理：协方差矩阵的特征向量 = 数据变化的主要方向
     - 特征值大小 = 该方向上的方差大小
     - 应用：保留大特征值对应的特征向量，丢弃小特征值方向
     - 效果：用少数主成分表示大部分数据信息

  2. **推荐系统**：
     - 用户-物品评分矩阵的特征分解
     - 特征向量 = 用户/物品的潜在因子
     - 特征值 = 该因子的重要性
     - 应用：矩阵补全、协同过滤

  3. **图像压缩**：
     - 图像矩阵的特征分解
     - 保留主要特征值对应的特征向量
     - 用低秩近似表示图像
     - 效果：大幅减少存储空间

  4. **PageRank算法**：
     - 网页链接矩阵的特征向量
     - 主特征向量 = 网页重要性排名
     - Google搜索引擎的核心算法

  5. **谱聚类**：
     - 相似度矩阵的特征分解
     - 特征向量 = 数据的低维嵌入
     - 在低维空间进行聚类

#### 7. 矩阵乘法在神经网络中
- 层间计算：y = W×x + b
- W：权重矩阵，x：输入向量，b：偏置向量，y：输出向量

**AI应用理解：**
- **特征向量**：数据表示、数据的主要变化方向
- **矩阵乘法**：神经网络层间计算、特征变换
- **转置**：数据变换、维度匹配

# 概率计算
# 抛硬币概率
def coin_probability(heads, tails):
    total = heads + tails
    p_heads = heads / total
    p_tails = tails / total
    return p_heads, p_tails

p_h, p_t = coin_probability(5, 5)
print(f"公平硬币概率：正面={p_h:.2f}，反面={p_t:.2f}")

# 贝叶斯定理应用
# 疾病检测问题
def bayes_theorem(p_disease, p_positive_given_disease, p_positive_given_healthy):
    p_positive = (p_positive_given_disease * p_disease + 
                  p_positive_given_healthy * (1 - p_disease))
    p_disease_given_positive = (p_positive_given_disease * p_disease) / p_positive
    return p_disease_given_positive

# 已知条件
p_disease = 0.01  # 疾病患病率1%
p_positive_given_disease = 0.95  # 真阳性率95%
p_positive_given_healthy = 0.05  # 假阳性率5%

result = bayes_theorem(p_disease, p_positive_given_disease, p_positive_given_healthy)
print(f"检测阳性时患病概率：{result:.4f}")
```

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 正态分布
mu, sigma = 0, 1
normal_dist = stats.norm(mu, sigma)

# 生成样本
samples = normal_dist.rvs(1000)

# 计算统计量
mean_sample = np.mean(samples)
std_sample = np.std(samples)

print(f"样本均值：{mean_sample:.4f}")
print(f"样本标准差：{std_sample:.4f}")

# 可视化
plt.figure(figsize=(10, 6))
plt.hist(samples, bins=30, density=True, alpha=0.7, label='样本分布')

# 理论分布
x = np.linspace(-4, 4, 100)
y = normal_dist.pdf(x)
plt.plot(x, y, 'r-', label='理论分布')

plt.title('正态分布样本与理论分布对比')
plt.xlabel('值')
plt.ylabel('概率密度')
plt.legend()
plt.grid(True)
plt.show()

# 二项分布
n = 10  # 试验次数
p = 0.5  # 成功概率
binom_dist = stats.binom(n, p)

# 计算概率
k = 5  # 成功5次的概率
prob = binom_dist.pmf(k)
print(f"10次试验成功5次的概率：{prob:.4f}")

def numerical_derivative(f, x, h=1e-5):
    """计算函数在x处的数值导数"""
    return (f(x + h) - f(x - h)) / (2 * h)

# 测试函数
def f(x):
    return x**2 + 2*x + 1

# 计算导数
x = 2
derivative = numerical_derivative(f, x)
print(f"函数 f(x)=x²+2x+1 在 x={x} 处的导数：{derivative:.4f}")
print(f"理论导数：2*x+2 = {2*x+2}")

# 多元函数梯度
def gradient_2d(f, x, y, h=1e-5):
    """计算二元函数的梯度"""
    df_dx = (f(x + h, y) - f(x - h, y)) / (2 * h)
    df_dy = (f(x, y + h) - f(x, y - h)) / (2 * h)
    return np.array([df_dx, df_dy])

# 测试函数：f(x,y) = x² + y²
def f_2d(x, y):
    return x**2 + y**2

# 计算梯度
x, y = 1, 2
grad = gradient_2d(f_2d, x, y)
print(f"函数 f(x,y)=x²+y² 在点({x},{y})处的梯度：{grad}")
print(f"理论梯度：(2x, 2y) = ({2*x}, {2*y})")

import numpy as np
import matplotlib.pyplot as plt

# 梯度下降法实现
def gradient_descent(f, gradient, start, learning_rate=0.1, n_iterations=100):
    """梯度下降算法"""
    x = start
    history = [x]
    
    for i in range(n_iterations):
        grad = gradient(x)
        x = x - learning_rate * grad
        history.append(x)
    
    return x, history

# 测试函数：f(x) = x²

    return x**2

def gradient_f(x):
    return 2*x

import numpy as np

# 熵计算
def entropy(probabilities):
    """计算熵 H(p) = -∑ p(x) * log2(p(x))"""
    # 移除概率为0的项
    p = probabilities[probabilities > 0]
    return -np.sum(p * np.log2(p))

# 示例：抛硬币
fair_coin = np.array([0.5, 0.5])  # 公平硬币
biased_coin = np.array([0.9, 0.1])  # 偏硬币

H_fair = entropy(fair_coin)
H_biased = entropy(biased_coin)

print(f"公平硬币的熵：{H_fair:.4f}")
print(f"偏硬币的熵：{H_biased:.4f}")

# KL散度
def kl_divergence(p, q):
    """计算KL散度 D_KL(p||q) = ∑ p(x) * log(p(x)/q(x))"""
    # 移除概率为0的项
    mask = (p > 0) & (q > 0)
    p = p[mask]
    q = q[mask]
    return np.sum(p * np.log(p / q))

# 示例
p = np.array([0.5, 0.5])
q = np.array([0.9, 0.1])

kl = kl_divergence(p, q)
print(f"KL散度 D_KL(p||q)：{kl:.4f}")

# 交叉熵
def cross_entropy(p, q):
    """计算交叉熵 H(p,q) = -∑ p(x) * log(q(x))"""
    # 移除概率为0的项
    mask = (p > 0) & (q > 0)
    p = p[mask]
    q = q[mask]
    return -np.sum(p * np.log(q))

ce = cross_entropy(p, q)
print(f"交叉熵 H(p,q)：{ce:.4f}")
```
