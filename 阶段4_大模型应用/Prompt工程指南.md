# Prompt工程指南（3-4周）

## 学习目标
掌握提示词设计技巧，能够编写高质量的Prompt让LLM更好地完成任务。

---

## 什么是Prompt工程？

**一句话**：Prompt工程就是"如何问问题"，让AI给出更好的答案。

**类比**：
```
差的提问："帮我写点东西"
好的提问："帮我写一封正式的求职邮件，应聘产品经理职位，强调3年经验和数据分析能力"

同样的AI，不同的提问方式，结果天差地别
```

---

## 第一周：Prompt设计原则

### 1.1 基本原则

**清晰具体**：
```
❌ "写一篇关于AI的文章"
✅ "写一篇800字的文章，介绍AI在医疗领域的三个应用，包括诊断、药物研发和手术机器人"
```

**提供上下文**：
```
❌ "解释这个代码"
✅ "我是一个Python初学者，请用简单的语言解释这段代码的作用：..."
```

**指定格式**：
```
❌ "列出优点"
✅ "用5个要点列出优点，每个要点不超过20字"
```

### 1.2 Prompt结构模板

**万能模板**：
```
# 角色
你是一个{角色描述}

# 任务
请{具体任务}

# 要求
1. {要求1}
2. {要求2}
3. {要求3}

# 格式
{输出格式要求}

# 示例（可选）
输入：...
输出：...
```

**示例**：
```
# 角色
你是一个资深的数据分析师

# 任务
请分析这份销售数据，找出销售下滑的原因

# 要求
1. 使用通俗易懂的语言
2. 给出至少3个可能的原因
3. 每个原因都要有数据支持

# 格式
用Markdown列表格式输出
```

## 第二周：零样本与少样本学习

### 2.1 零样本学习（Zero-Shot）

**定义**：不给任何例子，直接让模型完成任务

```
Prompt：
请将以下英文翻译成中文：
"Hello, how are you?"

AI：你好，你好吗？
```

**适用场景**：
- 简单任务（翻译、总结）
- 模型已经擅长的任务

**技巧**：
```
添加任务描述：
❌ "Hello, how are you?"
✅ "请将以下英文翻译成中文：Hello, how are you?"
```

### 2.2 少样本学习（Few-Shot）

**定义**：给几个例子，让模型模仿

```
Prompt：
请将以下英文翻译成中文：

示例1：
EN: Good morning
CN: 早上好

示例2：
EN: Thank you very much
CN: 非常感谢

现在翻译：
EN: How are you today?
CN: 

AI：你今天好吗？
```

**示例数量选择**：
- 简单任务：1-2个例子
- 复杂任务：3-5个例子
- 注意token限制

**示例质量**：
```
✅ 示例要准确
✅ 示例要多样化
✅ 示例要和目标任务相似
```

## 第三周：思维链（Chain of Thought）

### 3.1 什么是思维链？

**一句话**：让模型"一步步思考"，而不是直接给答案。

**类比**：
```
普通回答：直接写答案
思维链：写出解题过程

就像数学题，写出过程比只写答案得分更高
```

**示例**：
```
❌ 普通Prompt：
问题：一个农场有15头牛，每头牛每天产20升牛奶，一周能产多少升？
答案：2100升

✅ 思维链Prompt：
问题：一个农场有15头牛，每头牛每天产20升牛奶，一周能产多少升？
请一步步思考：

答案：
1. 每天产奶量 = 15头牛 × 20升/头 = 300升
2. 一周产奶量 = 300升/天 × 7天 = 2100升
所以一周能产2100升牛奶。
```

### 3.2 思维链技巧

**添加"让我们一步步思考"**：
```
Prompt：
问题：...
请一步步思考，然后给出答案。
```

**添加推理步骤要求**：
```
Prompt：
问题：...
请按照以下步骤回答：
1. 理解问题
2. 列出已知信息
3. 进行推理
4. 得出结论
```

**自我验证**：
```
Prompt：
问题：...
请一步步思考，然后检查你的答案是否正确。
```

## 第四周：高级技巧

### 4.1 角色扮演

**让AI扮演特定角色**：
```
Prompt：
你是一位经验丰富的医生。请用通俗易懂的语言解释高血压的成因和预防方法。
```

**效果**：
- 回答更专业
- 语气更恰当
- 内容更聚焦

### 4.2 系统提示（System Prompt）

**定义**：设定AI的行为规则

```
System Prompt：
你是一个友好的AI助手。
- 始终用中文回答
- 回答要简洁明了
- 如果不确定，坦诚说明
- 不要编造信息

User：什么是机器学习？
AI：机器学习是人工智能的一个分支，让计算机从数据中学习规律，而不需要明确编程。
```

### 4.3 负面提示

**告诉AI不要做什么**：
```
Prompt：
请总结这篇文章。
注意：
- 不要超过200字
- 不要使用专业术语
- 不要添加个人观点
```

### 4.4 输出格式控制

**JSON格式**：
```
Prompt：
请分析以下文本的情感，并以JSON格式返回：
{
  "sentiment": "positive/negative/neutral",
  "confidence": 0.95,
  "reason": "原因说明"
}

文本：这部电影真好看！
```

**Markdown格式**：
```
Prompt：
请用Markdown格式总结机器学习的三大类型：
- 使用标题
- 使用列表
- 每个类型用1-2句话说明
```

### 案例1：文本总结

```
Prompt：

1. 不超过100字
2. 提取3个关键点
3. 使用通俗语言

文章内容：
{文章内容}
```

### 案例2：代码生成

```
Prompt：
请帮我写一个Python函数，要求：
- 函数名：calculate_fibonacci
- 输入：n（整数）
- 输出：第n个斐波那契数
- 包含类型注解
- 包含docstring

示例：
calculate_fibonacci(5) → 5
calculate_fibonacci(10) → 55

## 常见问题

### Q1：Prompt越长越好吗？

**不一定**：
- 简单任务：短prompt即可
- 复杂任务：需要详细prompt
- 注意token限制

### Q2：为什么AI不按格式回答？

**解决方法**：
```
1. 更明确指定格式
2. 给出示例
3. 使用分隔符（如---、```）
4. 检查是否超出token限制
```

### Q3：如何提高回答质量？

**技巧**：
```
1. 给AI"思考时间"（思维链）
2. 提供上下文
3. 指定角色
4. 要求自我验证
5. 使用少样本学习
```
