## 学习目标
掌握大模型微调技术，能够使用LoRA、QLoRA等方法微调模型。

## 什么是模型微调？

**一句话**：在预训练模型基础上，用特定数据进一步训练，让它更擅长特定任务。

**类比**：
```
预训练 = 大学毕业（通用知识）
微调 = 职业培训（专业技能）

通用GPT → 医疗GPT（医疗数据微调）
通用GPT → 代码GPT（代码数据微调）
```

**为什么要微调？**
- Prompt工程有局限性
- 某些任务需要专业能力
- 需要符合特定风格/语气
- 需要处理私有数据

---

## 第一周：微调方法概述

### 1.1 微调方法分类

```
全参数微调：调整所有参数（最耗资源）
  ↓
参数高效微调（PEFT）：只调整部分参数
  ├─ LoRA：添加低秩矩阵
  ├─ Adapter：添加小型网络
  └─ Prefix Tuning：添加可学习前缀
```

### 1.2 方法对比

| 方法 | 参数量 | 显存需求 | 效果 | 适用场景 |
|------|--------|----------|------|----------|
| 全参数微调 | 100% | 极高 | 最好 | 资源充足 |
| LoRA | 0.1%-1% | 低 | 好 | 通用场景 |
| QLoRA | 0.1%-1% | 更低 | 好 | 显存有限 |
| Adapter | 1%-5% | 中 | 中等 | 多任务 |
| Prefix Tuning | <1% | 低 | 中等 | 生成任务 |

## 第二周：LoRA详解

### 2.1 什么是LoRA？

**全称**：Low-Rank Adaptation（低秩适配）

**核心思想**：大模型的参数更新是"低秩"的，可以用小矩阵近似

```
原始权重：W (d×d)  →  非常大
LoRA：W + A×B      →  A(d×r), B(r×d)，r很小

示例：
d=4096, r=16
原始：4096×4096 = 1600万参数
LoRA：4096×16 + 16×4096 = 13万参数（仅0.8%）
```

**类比**：
```
全参数微调：重新装修整个房子
LoRA：只换窗帘和沙发套，效果差不多但省很多钱
```

### 2.2 LoRA原理图解

```
原始权重 W (冻结)
    ↓
    + ─── LoRA路径
    │     A (d×r) → B (r×d)
    │     ↓
    └─────+
          ↓
       输出 = W×x + A×B×x
```

### 2.3 LoRA代码实现

```python
from peft import LoraConfig, get_peft_model, TaskType
from transformers import AutoModelForCausalLM

# 加载基础模型
model = AutoModelForCausalLM.from_pretrained("meta-llama/Llama-2-7b-hf")

# LoRA配置
lora_config = LoraConfig(
    task_type=TaskType.CAUSAL_LM,
    r=16,  # LoRA秩
    lora_alpha=32,  # 缩放因子
    lora_dropout=0.1,
    target_modules=["q_proj", "v_proj"]  # 应用到哪些层
)

# 应用LoRA
model = get_peft_model(model, lora_config)

# 查看可训练参数
model.print_trainable_parameters()
# 输出：trainable params: 4,194,304 || all params: 6,742,609,920 || trainable%: 0.0622
```

### 2.4 LoRA超参数选择

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| r | 8-64 | 秩越大，能力越强，但参数越多 |
| alpha | 2×r | 通常设为r的2倍 |
| dropout | 0.05-0.1 | 防止过拟合 |
| target_modules | q_proj, v_proj | 注意力层 |

## 第三周：QLoRA

### 3.1 什么是QLoRA？

**一句话**：4-bit量化 + LoRA，显存需求更低

**核心创新**：
1. 4-bit NormalFloat量化
2. 双重量化（量化常数也被量化）
3. 分页优化器

**效果**：
```
LLaMA-7B：
- 全精度：14GB显存
- 4-bit：4GB显存
- QLoRA：5GB显存（可训练）
```

### 3.2 QLoRA代码实现

```python
from transformers import AutoModelForCausalLM, BitsAndBytesConfig
from peft import LoraConfig, get_peft_model

# 量化配置
bnb_config = BitsAndBytesConfig(
    load_in_4bit=True,  # 4-bit量化
    bnb_4bit_quant_type="nf4",  # NormalFloat4
    bnb_4bit_compute_dtype=torch.float16,
    bnb_4bit_use_double_quant=True  # 双重量化
)

# 加载量化模型
model = AutoModelForCausalLM.from_pretrained(
    "meta-llama/Llama-2-7b-hf",
    quantization_config=bnb_config,
    device_map="auto"
)

# LoRA配置
lora_config = LoraConfig(
    r=16,
    lora_alpha=32,
    target_modules=["q_proj", "v_proj"],
    lora_dropout=0.1,
    bias="none"
)

# 应用QLoRA
model = get_peft_model(model, lora_config)
```

## 第四周：训练流程

### 4.1 完整微调流程

```
1. 准备数据
   ↓
2. 加载模型和Tokenizer
   ↓
3. 配置LoRA/QLoRA
   ↓
4. 设置训练参数
   ↓
5. 训练
   ↓
6. 保存模型
   ↓
7. 测试和部署
```

### 4.2 数据准备

**数据格式**：
```json
{
  "instruction": "将以下英文翻译成中文",
  "input": "Hello, how are you?",
  "output": "你好，你好吗？"
}
```

**数据处理代码**：
```python
from datasets import load_dataset

# 加载数据集
dataset = load_dataset("json", data_files="train.json")

# 格式化为prompt
def format_prompt(example):
    if example["input"]:
        return f"""### Instruction:
{example["instruction"]}

### Input:
{example["input"]}

### Response:
{example["output"]}"""
    else:
        return f"""### Instruction:
{example["instruction"]}

### Response:
{example["output"]}"""

dataset = dataset.map(lambda x: {"text": format_prompt(x)})
```

### 4.3 训练配置

```python
from transformers import TrainingArguments

training_args = TrainingArguments(
    output_dir="./results",
    num_train_epochs=3,
    per_device_train_batch_size=4,
    gradient_accumulation_steps=4,
    learning_rate=2e-4,
    warmup_steps=100,
    logging_steps=10,
    save_strategy="epoch",
    evaluation_strategy="epoch",
    fp16=True,
    optim="paged_adamw_8bit"  # 8-bit优化器
)
```

### 4.4 训练代码

```python
from transformers import Trainer

trainer = Trainer(
    model=model,
    args=training_args,
    train_dataset=dataset["train"],
    data_collator=DataCollatorForLanguageModeling(tokenizer, mlm=False)
)

# 开始训练
trainer.train()

# 保存LoRA权重
model.save_pretrained("./lora_weights")
```

### 4.5 合并与推理

```python
from peft import PeftModel

# 加载基础模型
base_model = AutoModelForCausalLM.from_pretrained("meta-llama/Llama-2-7b-hf")

# 加载LoRA权重
model = PeftModel.from_pretrained(base_model, "./lora_weights")

# 合并权重（可选）
merged_model = model.merge_and_unload()
merged_model.save_pretrained("./merged_model")

# 推理
inputs = tokenizer("翻译成中文：Hello", return_tensors="pt")
outputs = model.generate(**inputs, max_length=100)
print(tokenizer.decode(outputs[0]))
```

## 第五周：RLHF与DPO

### 5.1 什么是RLHF？

**全称**：Reinforcement Learning from Human Feedback（人类反馈强化学习）

**核心思想**：让人类的偏好指导模型学习

```
传统：模型学习"预测下一个词"
RLHF：模型学习"生成人类喜欢的回答"

ChatGPT = 预训练 + 指令微调 + RLHF
```

### 5.2 RLHF流程

```
步骤1：指令微调（SFT）
  模型学习遵循指令
  ↓
步骤2：训练奖励模型（RM）
  人类标注：回答A > 回答B
  训练模型预测人类偏好
  ↓
步骤3：强化学习（PPO）
  用奖励模型指导生成
  生成更好的回答
```

### 5.3 什么是DPO？

**全称**：Direct Preference Optimization（直接偏好优化）

**核心思想**：跳过奖励模型，直接学习偏好

```
RLHF：偏好数据 → 奖励模型 → 强化学习
DPO：偏好数据 → 直接优化

DPO更简单，效果相当
```

### 5.4 DPO代码实现

```python
from trl import DPOTrainer, DPOConfig

# 偏好数据格式
# {"prompt": "问题", "chosen": "好的回答", "rejected": "差的回答"}

dpo_config = DPOConfig(
    beta=0.1,  # 温度参数
    learning_rate=5e-5,
    num_train_epochs=3
)

trainer = DPOTrainer(
    model=model,
    ref_model=ref_model,  # 参考模型
    args=dpo_config,
    train_dataset=dataset,
    tokenizer=tokenizer
)

trainer.train()
```

## 微调实战案例

### 案例：训练一个客服助手

**数据准备**：
```json
[
  {
    "instruction": "回答用户关于退货的问题",
    "input": "我买的东西不满意，可以退货吗？",
    "output": "您好！我们支持7天无理由退货。请您登录账户，在订单详情页申请退货，我们会尽快处理。如有其他问题，请随时联系我们。"
  }
]
```

**训练代码**：
```python
# 1. 加载模型
model_name = "Qwen/Qwen2-7B"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model = AutoModelForCausalLM.from_pretrained(model_name, load_in_4bit=True)

# 2. 配置QLoRA
lora_config = LoraConfig(r=16, target_modules=["q_proj", "v_proj"])
model = get_peft_model(model, lora_config)

# 3. 训练
trainer = Trainer(model=model, args=training_args, train_dataset=dataset)
trainer.train()

# 4. 测试
prompt = "### Instruction:\n回答用户关于退货的问题\n\n### Input:\n我买的东西不满意，可以退货吗？\n\n### Response:\n"
inputs = tokenizer(prompt, return_tensors="pt")
outputs = model.generate(**inputs, max_length=200)
print(tokenizer.decode(outputs[0]))
```
