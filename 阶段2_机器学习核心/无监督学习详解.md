## 无监督学习概述
无监督学习是从**无标签数据**中发现数据的内在结构和模式。主要任务包括：
- **聚类**：将相似的数据点分组
- **降维**：减少数据维度同时保留重要信息
- **异常检测**：识别与大多数数据不同的异常点
- **关联规则**：发现数据项之间的关联关系

## 第一周：聚类算法

### 1.1 K-Means聚类

#### 原理
K-Means通过迭代优化将数据划分为K个簇，使每个簇内数据点到簇中心的距离最小。

#### 算法流程

```
输入：数据集X，簇数K
输出：K个簇的划分

1. 随机初始化K个簇中心 μ₁, μ₂, ..., μₖ
2. 重复以下步骤直到收敛：
   a. 分配步骤：将每个点分配到最近的簇中心
      Cᵢ = {xₚ : ||xₚ - μᵢ||² ≤ ||xₚ - μⱼ||², ∀j}
   b. 更新步骤：重新计算每个簇的中心
      μᵢ = (1/|Cᵢ|) ∑_{x∈Cᵢ} x
3. 返回簇划分C₁, C₂, ..., Cₖ
```

#### 目标函数（失真函数）
```
J = ∑ᵢ₌₁ᴷ ∑_{x∈Cᵢ} ||x - μᵢ||²
```

#### 优缺点

**优点**：
- 简单易懂，实现方便
- 计算效率高，适合大数据集
- 收敛速度快

**缺点**：
- 需要预先指定K值
- 对初始中心敏感
- 只能发现球形簇
- 对噪声和异常值敏感

#### K值选择方法

**肘部法则（Elbow Method）**：
- 计算不同K值下的J值
- 选择J下降速度变缓的拐点

**轮廓系数（Silhouette Coefficient）**：
```
s(i) = (b(i) - a(i)) / max(a(i), b(i))
```
- a(i)：样本i到同簇其他样本的平均距离
- b(i)：样本i到最近其他簇的平均距离
- s(i) ∈ [-1, 1]，越大越好

#### Python实现
```python
import numpy as np
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score
from sklearn.datasets import make_blobs
import matplotlib.pyplot as plt

# 生成模拟数据
X, y_true = make_blobs(n_samples=300, centers=4, cluster_std=0.60, random_state=42)

# 肘部法则选择K
K_range = range(2, 11)
inertias = []
silhouette_scores = []

for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
    kmeans.fit(X)
    inertias.append(kmeans.inertia_)
    silhouette_scores.append(silhouette_score(X, kmeans.labels_))

# 可视化
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

ax1.plot(K_range, inertias, 'bo-')
ax1.set_xlabel('K值')
ax1.set_ylabel('惯性（Inertia）')
ax1.set_title('肘部法则')

ax2.plot(K_range, silhouette_scores, 'ro-')
ax2.set_xlabel('K值')
ax2.set_ylabel('轮廓系数')
ax2.set_title('轮廓系数法')

plt.tight_layout()
plt.show()

# 使用最优K值训练
best_k = K_range[np.argmax(silhouette_scores)]
kmeans = KMeans(n_clusters=best_k, random_state=42, n_init=10)
labels = kmeans.fit_predict(X)

print(f"最优K值: {best_k}")
print(f"轮廓系数: {silhouette_score(X, labels):.4f}")
```

### 1.2 层次聚类

#### 原理
层次聚类通过逐步合并或分裂簇来构建聚类的层次结构。

#### 两种方法

**凝聚型（自底向上）**：
1. 将每个样本作为一个簇
2. 重复合并最近的两个簇
3. 直到达到指定簇数或满足停止条件

**分裂型（自顶向下）**：
1. 将所有样本作为一个簇
2. 重复分裂最不纯的簇
3. 直到达到指定簇数

#### 距离度量

**簇间距离**：
- **单链接（Single Linkage）**：两簇间最近点的距离
  ```
  d(Cᵢ, Cⱼ) = min_{x∈Cᵢ, y∈Cⱼ} d(x, y)
  ```
- **全链接（Complete Linkage）**：两簇间最远点的距离
  ```
  d(Cᵢ, Cⱼ) = max_{x∈Cᵢ, y∈Cⱼ} d(x, y)
  ```
- **平均链接（Average Linkage）**：两簇间所有点对的平均距离
  ```
  d(Cᵢ, Cⱼ) = (1/|Cᵢ||Cⱼ|) ∑_{x∈Cᵢ} ∑_{y∈Cⱼ} d(x, y)
  ```
- **Ward方法**：最小化合并后的方差增量

#### Python实现
```python
from sklearn.cluster import AgglomerativeClustering
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt

# 层次聚类
hierarchical = AgglomerativeClustering(n_clusters=4, linkage='ward')
labels = hierarchical.fit_predict(X)

# 绘制树状图
plt.figure(figsize=(12, 5))
Z = linkage(X, method='ward')
dendrogram(Z, truncate_mode='lastp', p=30)
plt.title('层次聚类树状图')
plt.xlabel('样本索引')
plt.ylabel('距离')
plt.show()
```

### 1.3 DBSCAN

#### 原理
DBSCAN（Density-Based Spatial Clustering of Applications with Noise）基于密度进行聚类，能发现任意形状的簇并识别噪声点。

#### 核心概念

- **ε邻域**：点p周围半径ε内的所有点
- **核心点**：ε邻域内至少有MinPts个点
- **边界点**：在核心点的ε邻域内，但自身不是核心点
- **噪声点**：既不是核心点也不是边界点

#### 算法流程

```
输入：数据集D，半径ε，最小点数MinPts
输出：簇划分

1. 对于每个点p：
   a. 如果p是核心点：
      - 创建新簇C
      - 将p加入C
      - 将p的ε邻域内所有点加入C
      - 递归扩展C（将邻域内核心点的邻域也加入）
   b. 如果p不是核心点且未被访问：标记为噪声

2. 返回所有簇
```

#### 参数选择

**ε选择**：
- 计算每个点到第k个最近邻的距离
- 绘制距离排序图，找拐点

**MinPts选择**：
- 经验值：MinPts ≥ 维度 + 1
- 通常取 MinPts = 2 × 维度

#### 优缺点

**优点**：
- 不需要预先指定簇数
- 能发现任意形状的簇
- 能识别噪声点
- 对异常值不敏感

**缺点**：
- 对参数ε和MinPts敏感
- 对密度不均匀的数据效果差
- 高维数据效果不好

#### Python实现
```python
from sklearn.cluster import DBSCAN
from sklearn.neighbors import NearestNeighbors
import numpy as np

# 使用k距离图选择ε
k = 5
neighbors = NearestNeighbors(n_neighbors=k)
neighbors.fit(X)
distances, _ = neighbors.kneighbors(X)
k_distances = np.sort(distances[:, -1])

plt.figure(figsize=(10, 5))
plt.plot(k_distances)
plt.xlabel('点（按距离排序）')
plt.ylabel(f'{k}近邻距离')
plt.title('k距离图（用于选择ε）')
plt.grid(True)
plt.show()

# DBSCAN聚类
dbscan = DBSCAN(eps=0.5, min_samples=5)
labels = dbscan.fit_predict(X)

# 统计结果
n_clusters = len(set(labels)) - (1 if -1 in labels else 0)
n_noise = list(labels).count(-1)

print(f"发现的簇数: {n_clusters}")
print(f"噪声点数: {n_noise}")
print(f"轮廓系数: {silhouette_score(X[labels != -1], labels[labels != -1]):.4f}")
```

## 第二周：降维技术

### 2.1 PCA（主成分分析）

#### 原理
PCA通过线性变换将数据投影到方差最大的方向上，实现降维。

#### 数学推导

**目标**：找到使投影后方差最大的方向

**步骤**：
1. 数据中心化：X̄ = X - mean(X)
2. 计算协方差矩阵：C = (1/n) X̄ᵀX̄
3. 特征值分解：C = PDPᵀ
4. 选择前d个最大特征值对应的特征向量
5. 投影：X_new = X̄ × P[:,:d]

**方差解释率**：
```
解释率 = ∑ᵢ₌₁ᵈ λᵢ / ∑ᵢ₌₁ᴰ λᵢ
```

#### 优缺点

**优点**：
- 简单有效，计算快
- 可以去除噪声
- 便于可视化（降到2D或3D）

**缺点**：
- 只能捕捉线性关系
- 对异常值敏感
- 主成分可能难以解释

#### Python实现
```python
from sklearn.decomposition import PCA
from sklearn.datasets import load_digits
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt

# 加载数据
digits = load_digits()
X = digits.data
y = digits.target

# 标准化
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# PCA降维
pca = PCA()
pca.fit(X_scaled)

# 方差解释率
cumsum = np.cumsum(pca.explained_variance_ratio_)
d = np.argmax(cumsum >= 0.95) + 1  # 保留95%方差

print(f"原始维度: {X.shape[1]}")
print(f"保留95%方差需要: {d}个主成分")

# 可视化方差解释率
plt.figure(figsize=(10, 5))
plt.plot(cumsum, 'b-')
plt.axhline(y=0.95, color='r', linestyle='--', label='95%方差')
plt.axvline(x=d, color='g', linestyle='--', label=f'd={d}')
plt.xlabel('主成分数')
plt.ylabel('累积方差解释率')
plt.title('PCA方差解释率')
plt.legend()
plt.grid(True)
plt.show()

# 降到2维可视化
pca_2d = PCA(n_components=2)
X_2d = pca_2d.fit_transform(X_scaled)

plt.figure(figsize=(10, 8))
scatter = plt.scatter(X_2d[:, 0], X_2d[:, 1], c=y, cmap='tab10', s=10)
plt.colorbar(scatter)
plt.xlabel('PC1')
plt.ylabel('PC2')
plt.title('PCA降维到2维')
plt.show()
```

### 2.2 t-SNE

#### 原理
t-SNE（t-Distributed Stochastic Neighbor Embedding）是一种非线性降维方法，特别适合高维数据可视化。

#### 核心思想

**高维空间相似度**：
```
pⱼ|ᵢ = exp(-||xᵢ - xⱼ||²/2σᵢ²) / ∑ₖ≠ᵢ exp(-||xᵢ - xₖ||²/2σᵢ²)
```

**低维空间相似度**：
```
qᵢⱼ = (1 + ||yᵢ - yⱼ||²)⁻¹ / ∑ₖ≠ₗ (1 + ||yₖ - yₗ||²)⁻¹
```

**优化目标（KL散度）**：
```
KL(P||Q) = ∑ᵢ ∑ⱼ pᵢⱼ log(pᵢⱼ/qᵢⱼ)
```

#### 优缺点

**优点**：
- 能保留局部结构
- 可视化效果好
- 能发现非线性结构

**缺点**：
- 计算复杂度高
- 结果依赖超参数
- 不同运行结果可能不同
- 只能用于可视化（不能用于其他任务）

#### Python实现
```python
from sklearn.manifold import TSNE

# t-SNE降维
tsne = TSNE(n_components=2, random_state=42, perplexity=30, n_iter=1000)
X_tsne = tsne.fit_transform(X_scaled)

# 可视化
plt.figure(figsize=(10, 8))
scatter = plt.scatter(X_tsne[:, 0], X_tsne[:, 1], c=y, cmap='tab10', s=10)
plt.colorbar(scatter)
plt.xlabel('t-SNE 1')
plt.ylabel('t-SNE 2')
plt.title('t-SNE降维可视化')
plt.show()
```

### 2.3 UMAP

#### 原理
UMAP（Uniform Manifold Approximation and Projection）基于拓扑数据分析理论，在保持全局结构的同时实现降维。

#### 与t-SNE对比

| 特性 | t-SNE | UMAP |
|------|-------|------|
| 速度 | 慢 | 快 |
| 全局结构 | 保留较差 | 保留较好 |
| 可复现性 | 差 | 好 |
| 可用于新数据 | 否 | 是 |

#### Python实现
```python
# 需要安装umap-learn: pip install umap-learn
import umap

# UMAP降维
reducer = umap.UMAP(n_components=2, random_state=42)
X_umap = reducer.fit_transform(X_scaled)

# 可视化
plt.figure(figsize=(10, 8))
scatter = plt.scatter(X_umap[:, 0], X_umap[:, 1], c=y, cmap='tab10', s=10)
plt.colorbar(scatter)
plt.xlabel('UMAP 1')
plt.ylabel('UMAP 2')
plt.title('UMAP降维可视化')
plt.show()
```

## 第三周：异常检测

### 3.1 基于统计的方法

#### Z-Score方法
```python
from scipy import stats

def detect_outliers_zscore(data, threshold=3):
    """使用Z-Score检测异常值"""
    z_scores = np.abs(stats.zscore(data))
    return z_scores > threshold
```

#### IQR方法
```python
def detect_outliers_iqr(data):
    """使用IQR检测异常值"""
    Q1 = np.percentile(data, 25)
    Q3 = np.percentile(data, 75)
    IQR = Q3 - Q1
    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR
    return (data < lower_bound) | (data > upper_bound)
```

### 3.2 Isolation Forest

#### 原理
Isolation Forest通过随机划分数据来"隔离"异常点，异常点通常需要更少的划分次数。

#### 核心思想
- 异常点是"容易被隔离"的点
- 随机选择特征和划分值
- 异常点的路径长度通常较短

#### 异常分数
```
s(x, n) = 2^(-E(h(x))/c(n))
```
- h(x)：点x的路径长度
- c(n)：平均路径长度的归一化因子
- s接近1：异常
- s接近0.5：正常

#### Python实现
```python
from sklearn.ensemble import IsolationForest
from sklearn.datasets import make_blobs

# 生成数据（包含异常点）
X_normal, _ = make_blobs(n_samples=300, centers=1, cluster_std=1.0, random_state=42)
X_outliers = np.random.uniform(low=-10, high=10, size=(20, 2))
X = np.vstack([X_normal, X_outliers])

# Isolation Forest
iso_forest = IsolationForest(contamination=0.1, random_state=42)
labels = iso_forest.fit_predict(X)
scores = iso_forest.decision_function(X)

# 可视化
plt.figure(figsize=(10, 6))
plt.scatter(X[labels == 1, 0], X[labels == 1, 1], c='blue', label='正常点')
plt.scatter(X[labels == -1, 0], X[labels == -1, 1], c='red', label='异常点')
plt.title('Isolation Forest异常检测')
plt.legend()
plt.show()

print(f"检测到的异常点数: {(labels == -1).sum()}")
```

### 3.3 Local Outlier Factor (LOF)

#### 原理
LOF通过比较点的局部密度与其邻居的局部密度来检测异常点。

#### 核心概念

**k距离**：点p到第k个最近邻的距离

**可达距离**：
```
reach-distₖ(p, o) = max(k-distance(o), d(p, o))
```

**局部可达密度**：
```
lrdₖ(p) = |Nₖ(p)| / ∑_{o∈Nₖ(p)} reach-distₖ(p, o)
```

**LOF分数**：
```
LOFₖ(p) = ∑_{o∈Nₖ(p)} lrdₖ(o) / (|Nₖ(p)| × lrdₖ(p))
```
- LOF ≈ 1：正常
- LOF > 1：可能是异常

#### Python实现
```python
from sklearn.neighbors import LocalOutlierFactor

# LOF异常检测
lof = LocalOutlierFactor(n_neighbors=20, contamination=0.1)
labels = lof.fit_predict(X)
scores = lof.negative_outlier_factor_

# 可视化
plt.figure(figsize=(10, 6))
plt.scatter(X[labels == 1, 0], X[labels == 1, 1], c='blue', label='正常点')
plt.scatter(X[labels == -1, 0], X[labels == -1, 1], c='red', label='异常点')
plt.title('LOF异常检测')
plt.legend()
plt.show()
```

## 算法选择指南

### 聚类算法对比

| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| K-Means | 简单快速 | 需要指定K，球形簇 | 大数据集 |
| 层次聚类 | 不需要K，层次结构 | 计算复杂度高 | 小数据集，需要层次 |
| DBSCAN | 任意形状，识别噪声 | 参数敏感 | 非球形簇，有噪声 |

### 降维算法对比

| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| PCA | 快速，可解释 | 只能线性 | 通用降维 |
| t-SNE | 可视化好 | 慢，不可复现 | 可视化 |
| UMAP | 快，保留全局结构 | 较新 | 可视化，通用 |

### 异常检测算法对比

| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| Isolation Forest | 快速，高维有效 | 不适合局部异常 | 通用 |
| LOF | 检测局部异常 | 计算复杂 | 密度不均匀 |
