# 阶段2：机器学习核心（2-3个月）

## 阶段目标
掌握机器学习核心算法和实践技能，能够独立解决实际问题。

## 学习时间
- **总时长**：2-3个月（根据基础调整）
- **每周投入**：15-20小时
- **建议节奏**：理论+实践结合，每周完成1-2个项目

---

## 学习内容概览

### 1. 监督学习（3-4周）

#### 回归算法
- **线性回归**：最小二乘法、梯度下降
- **正则化回归**：Ridge(L2)、Lasso(L1)、ElasticNet
- **多项式回归**：处理非线性关系

#### 分类算法
- **逻辑回归**：Sigmoid函数、交叉熵损失
- **决策树**：信息增益、基尼不纯度、剪枝
- **随机森林**：Bagging、随机特征选择
- **梯度提升**：GBDT、XGBoost、LightGBM

#### 距离与间隔模型
- **KNN**：距离度量、K值选择
- **SVM**：最大间隔、核技巧

#### 集成学习
- **Bagging**：并行训练、降低方差
- **Boosting**：串行训练、降低偏差
- **Stacking**：多层模型融合

### 2. 无监督学习（2-3周）

#### 聚类算法
- **K-Means**：迭代优化、K值选择
- **层次聚类**：凝聚型、分裂型
- **DBSCAN**：密度聚类、噪声识别

#### 降维技术
- **PCA**：主成分分析、方差最大化
- **t-SNE**：非线性降维、可视化
- **UMAP**：拓扑保持、高效降维

#### 异常检测
- **统计方法**：Z-Score、IQR
- **Isolation Forest**：孤立森林
- **LOF**：局部异常因子

### 3. 模型评估与优化（2-3周）

#### 评估指标
- **回归指标**：MSE、RMSE、MAE、R²
- **分类指标**：Accuracy、Precision、Recall、F1、AUC-ROC
- **聚类指标**：轮廓系数、Calinski-Harabasz

#### 交叉验证
- **K折交叉验证**：标准K折、分层K折
- **留一法**：LOOCV
- **时间序列**：滚动窗口验证

#### 超参数调优
- **网格搜索**：穷举搜索
- **随机搜索**：随机采样
- **贝叶斯优化**：智能搜索

#### 特征工程
- **特征缩放**：标准化、归一化
- **特征编码**：独热编码、标签编码
- **特征选择**：过滤法、包裹法、嵌入法

### 4. 实践项目（3-4周）

#### 回归项目
- 房价预测
- 销量预测
- 股票价格预测

#### 分类项目
- 客户流失预测
- 信用评分
- 垃圾邮件分类

#### 聚类项目
- 客户分群
- 图像分割
- 文档聚类

#### 综合项目
- 端到端机器学习流程
- 模型部署与服务化

## 算法选择指南

### 问题类型选择

```
监督学习?
├── 是
│   ├── 目标变量连续? → 回归问题
│   │   ├── 数据量小 → 线性回归、Ridge
│   │   ├── 数据量中 → 随机森林、GBDT
│   │   └── 数据量大 → XGBoost、LightGBM
│   └── 目标变量离散? → 分类问题
│       ├── 需要概率输出 → 逻辑回归
│       ├── 需要可解释性 → 决策树
│       └── 追求高精度 → 随机森林、XGBoost
└── 否
    ├── 需要分组? → 聚类
    │   ├── 知道簇数 → K-Means
    │   ├── 不知道簇数 → 层次聚类
    │   └── 有噪声 → DBSCAN
    ├── 需要降维? → PCA、t-SNE、UMAP
    └── 需要找异常? → Isolation Forest、LOF
```

### 算法对比表

| 算法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 线性回归 | 简单、可解释、快 | 线性假设 | 基准模型、特征少 |
| 逻辑回归 | 可解释、输出概率 | 线性边界 | 二分类、基准 |
| 决策树 | 可解释、非线性 | 过拟合 | 需要解释性 |
| 随机森林 | 精度高、鲁棒 | 慢、黑盒 | 通用 |
| XGBoost | 精度高、功能全 | 参数多 | 竞赛、工业 |
| SVM | 高维效果好 | 慢、调参难 | 小样本、高维 |
| KNN | 简单、无需训练 | 预测慢 | 小数据集 |
| K-Means | 快、简单 | 需要K、球形 | 大数据、球形簇 |
| DBSCAN | 任意形状、去噪 | 参数敏感 | 非球形、有噪声 |
