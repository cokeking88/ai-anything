# 阶段3：深度学习进阶（3-4个月）

## 阶段目标
掌握深度学习核心架构和PyTorch框架，能够构建和训练复杂的神经网络模型。

## 学习时间
- **总时长**：3-4个月
- **每周投入**：20-25小时
- **建议节奏**：理论+代码实现+项目实践

## 学习内容概览

### 1. 神经网络基础（3-4周）
- 感知机与多层感知机
- 激活函数：ReLU、Sigmoid、Tanh
- 损失函数：交叉熵、均方误差
- 优化器：SGD、Adam、RMSprop
- 反向传播算法
- 正则化技术：Dropout、BatchNorm

### 2. 卷积神经网络（3-4周）
- 卷积运算原理
- CNN架构演进：LeNet → AlexNet → VGG → ResNet
- 经典网络实现
- 目标检测基础：YOLO、Faster R-CNN
- 图像分割：U-Net、Mask R-CNN

### 3. 循环神经网络（2-3周）
- RNN原理与梯度消失问题
- LSTM与GRU
- 序列到序列模型
- 注意力机制基础
- 文本分类与情感分析

### 4. Transformer架构（3-4周）
- Self-Attention机制
- 多头注意力
- 位置编码
- BERT与GPT架构
- 预训练与微调

### 5. PyTorch框架（2-3周）
- 张量操作与自动微分
- 数据加载与预处理
- 模型定义与训练
- GPU加速与分布式训练
- 模型保存与加载

## 学习成果检验
- [ ] 理解深度学习核心概念
- [ ] 能够使用PyTorch构建复杂网络
- [ ] 掌握CNN、RNN、Transformer架构
- [ ] 完成3-5个深度学习项目
- [ ] 能够调优模型性能

4. **项目驱动**：通过项目巩固理论知识
