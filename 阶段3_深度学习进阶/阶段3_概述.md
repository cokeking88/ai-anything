# 阶段3：深度学习进阶（3-4个月）

## 阶段目标
掌握深度学习核心架构和PyTorch框架，能够构建和训练复杂的神经网络模型。

## 什么是深度学习？

**一句话**：深度学习 = 很多层的神经网络，能自动学习复杂模式

**类比**：
```
传统机器学习：手工提取特征 → 模型学习
深度学习：模型自动学习特征 + 学习规律

就像：
传统方法：老师告诉你重点 → 你背
深度学习：你自己看书 → 自己发现重点
```

## 学习内容概览

### 1. 神经网络基础（3-4周）

**核心问题**：神经网络是怎么工作的？

| 概念 | 通俗解释 |
|------|----------|
| 神经元 | 一个小型计算器，做加权求和 |
| 层 | 很多神经元排成一排 |
| 前向传播 | 数据从输入流向输出 |
| 反向传播 | 从错误中学习，调整权重 |
| 激活函数 | 决定神经元是否"激活" |
| 损失函数 | 衡量预测有多差 |

### 2. 卷积神经网络CNN（3-4周）

**核心问题**：如何让机器"看懂"图片？

| 概念 | 通俗解释 |
|------|----------|
| 卷积 | 用小窗口扫描图片，提取特征 |
| 池化 | 压缩图片，保留重要信息 |
| 特征图 | 卷积后的结果 |
| 经典架构 | LeNet → AlexNet → VGG → ResNet |

**应用**：图像分类、目标检测、人脸识别

### 3. 循环神经网络RNN（2-3周）

**核心问题**：如何让机器理解序列数据？

| 概念 | 通俗解释 |
|------|----------|
| RNN | 有记忆的神经网络 |
| 梯度消失 | 记不住长期依赖 |
| LSTM | 长短期记忆，解决梯度消失 |
| GRU | LSTM的简化版 |

**应用**：文本生成、机器翻译、语音识别

### 4. Transformer架构（3-4周）

**核心问题**：如何并行处理序列，捕捉全局依赖？

| 概念 | 通俗解释 |
|------|----------|
| 自注意力 | 每个词关注所有其他词 |
| 多头注意力 | 多个角度理解关系 |
| 位置编码 | 告诉模型词的顺序 |
| BERT/GPT | 预训练语言模型 |

**应用**：ChatGPT、BERT、机器翻译

### 5. PyTorch框架（2-3周）

**核心问题**：如何用代码实现深度学习？

| 概念 | 通俗解释 |
|------|----------|
| 张量 | 多维数组 |
| 自动微分 | 自动计算梯度 |
| nn.Module | 定义模型的基类 |
| DataLoader | 批量加载数据 |
