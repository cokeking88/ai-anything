## 什么是神经网络？

**一句话**：神经网络就是模仿人脑的计算模型，通过大量简单计算单元（神经元）的连接来学习复杂模式。

**生活类比**：
```
人脑：神经元 → 传递信号 → 学习
神经网络：计算单元 → 传递数据 → 学习

就像：
人脑通过不断练习学会骑车
神经网络通过不断调整参数学会预测
```

## 第一周：感知机与多层感知机

### 1.1 感知机：最简单的神经网络

#### 什么是感知机？

**直观理解**：感知机就是一个"投票器"，根据多个输入做出决定。

```
输入1（天气好）  ──┐
输入2（有时间）  ──┼──→ 加权求和 → 激活函数 → 输出（去/不去）
输入3（有钱）   ──┘

天气好×0.5 + 有时间×0.3 + 有钱×0.2 > 阈值 → 去！
```

**数学公式**：
```
y = f(w₁x₁ + w₂x₂ + ... + wₙxₙ + b)

w：权重（重要性）
x：输入
b：偏置（调整阈值）
f：激活函数
```

**类比**：
- 权重 = 每个因素有多重要
- 偏置 = 决定的难易程度
- 激活函数 = 最终拍板

#### 感知机能做什么？

**能做**：
- 线性可分的问题（用一条线/面能分开）
- 与门、或门

**不能做**：
- 非线性问题（如异或XOR）

```
与门（能做）：        异或（不能做）：
  1 |    ×              1 |  ×    ×
    |  ×                  |
  0 |×                  0 |    ×    ×
    +------              +--------
    0    1                0    1

能用直线分开          不能用直线分开
```

#### Python实现
```python
import numpy as np

class Perceptron:
    def __init__(self, n_inputs, learning_rate=0.01):
        self.weights = np.random.randn(n_inputs) * 0.01
        self.bias = 0
        self.lr = learning_rate
    
    def activation(self, z):
        """阶跃函数：大于0返回1，否则返回0"""
        return 1 if z >= 0 else 0
    
    def predict(self, x):
        z = np.dot(self.weights, x) + self.bias
        return self.activation(z)
    
    def train(self, X, y, epochs=100):
        for _ in range(epochs):
            for xi, yi in zip(X, y):
                y_pred = self.predict(xi)
                error = yi - y_pred
                # 更新权重
                self.weights += self.lr * error * xi
                self.bias += self.lr * error

# 训练与门
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([0, 0, 0, 1])  # 与门

p = Perceptron(2)
p.train(X, y)

print("与门测试：")
for xi, yi in zip(X, y):
    print(f"输入: {xi}, 预测: {p.predict(xi)}, 真实: {yi}")
```

### 1.2 多层感知机（MLP）

#### 为什么需要多层？

**问题**：单层感知机不能解决XOR问题

**解决**：叠加多层！

```
单层感知机：        多层感知机：

输入 → 输出         输入 → 隐藏层 → 输出

能处理线性问题      能处理非线性问题
```

#### MLP的结构

```
输入层        隐藏层1       隐藏层2       输出层
  ○ ────────→ ○ ────────→ ○ ────────→ ○
  ○ ────────→ ○ ────────→ ○ ────────→ ○
  ○ ────────→ ○ ────────→ ○
  ○ ────────→ ○

每层都有权重矩阵，每层输出是下层输入
```

**类比**：
- 输入层 = 看到数据
- 隐藏层 = 逐层抽象理解
- 输出层 = 做出判断

#### Python实现
```python
class MLP:
    def __init__(self, layer_sizes, learning_rate=0.01):
        """
        layer_sizes: [输入维度, 隐藏1, 隐藏2, ..., 输出维度]
        例：[2, 4, 1] 表示2输入，4隐藏，1输出
        """
        self.layer_sizes = layer_sizes
        self.lr = learning_rate
        self.weights = []
        self.biases = []
        
        # 初始化权重
        for i in range(len(layer_sizes) - 1):
            w = np.random.randn(layer_sizes[i], layer_sizes[i+1]) * 0.01
            b = np.zeros(layer_sizes[i+1])
            self.weights.append(w)
            self.biases.append(b)
    
    def sigmoid(self, z):
        return 1 / (1 + np.exp(-z))
    
    def forward(self, x):
        """前向传播：数据从输入流向输出"""
        self.activations = [x]
        
        for w, b in zip(self.weights, self.biases):
            z = np.dot(self.activations[-1], w) + b
            a = self.sigmoid(z)
            self.activations.append(a)
        
        return self.activations[-1]
    
    def train(self, X, y, epochs=1000):
        """简化版训练（省略反向传播细节）"""
        for epoch in range(epochs):
            output = self.forward(X)
            loss = np.mean((y - output) ** 2)
            
            if epoch % 100 == 0:
                print(f"Epoch {epoch}, Loss: {loss:.4f}")

# 测试XOR问题
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])  # XOR

mlp = MLP([2, 4, 1], learning_rate=0.5)
mlp.train(X, y, epochs=5000)

print("\nXOR预测：")
for xi, yi in zip(X, y):
    pred = mlp.forward(xi.reshape(1, -1))
    print(f"输入: {xi}, 真实: {yi[0]}, 预测: {pred[0][0]:.4f}")
```

## 第二周：激活函数与损失函数

### 2.1 激活函数

#### 什么是激活函数？

**直观理解**：激活函数决定神经元是否"激活"，引入非线性。

```
没有激活函数：多层 = 一层（线性叠加还是线性）
有激活函数：多层 ≠ 一层（非线性叠加）
```

#### 常用激活函数

**Sigmoid**：
```
        1 |      ___________
          |    /
        0 |___/
          +----------------→ x

输出范围：(0, 1)
用途：二分类输出层
缺点：梯度消失
```

**Tanh**：
```
        1 |        ___________
          |      /
        0 |_____/_______________
          |   /
       -1 |__/
          +----------------→ x

输出范围：(-1, 1)
优点：以0为中心
缺点：仍有梯度消失
```

**ReLU（最常用）**：
```
          |        /
          |      /
        0 |____/___________
          +----------------→ x

输出范围：[0, ∞)
优点：计算简单，缓解梯度消失
缺点：负数时梯度为0（神经元死亡）
```

**Leaky ReLU**：
```
          |        /
          |      /
          |    /
          |  /
          +----------------→ x

解决ReLU的神经元死亡问题
负数时有小的斜率（如0.01）
```

#### 激活函数选择指南

| 位置 | 推荐激活函数 |
|------|--------------|
| 隐藏层 | ReLU（首选） |
| 二分类输出 | Sigmoid |
| 多分类输出 | Softmax |
| 回归输出 | 无/线性 |

#### Python实现
```python
import numpy as np

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

def tanh(z):
    return np.tanh(z)

def relu(z):
    return np.maximum(0, z)

def leaky_relu(z, alpha=0.01):
    return np.where(z > 0, z, alpha * z)

def softmax(z):
    exp_z = np.exp(z - np.max(z))  # 数值稳定
    return exp_z / exp_z.sum(axis=-1, keepdims=True)
```

### 2.2 损失函数

#### 什么是损失函数？

**直观理解**：损失函数衡量预测有多"差"，训练目标是最小化损失。

```
预测：0.8，真实：1 → 损失小（预测接近）
预测：0.2，真实：1 → 损失大（预测差远了）
```

#### 回归损失函数

**均方误差（MSE）**：
```
L = (1/n) × Σ(预测 - 真实)²

对大误差惩罚重
最常用的回归损失
```

**平均绝对误差（MAE）**：
```
L = (1/n) × Σ|预测 - 真实|

对异常值鲁棒
```

#### 分类损失函数

**二元交叉熵**：
```
L = -[y×log(p) + (1-y)×log(1-p)]

y=1时：L = -log(p)，希望p接近1
y=0时：L = -log(1-p)，希望p接近0
```

**多分类交叉熵**：
```
L = -Σ yᵢ×log(pᵢ)

配合softmax使用
```

#### 损失函数选择指南

| 问题类型 | 推荐损失函数 |
|----------|--------------|
| 回归 | MSE |
| 回归（有异常值） | MAE / Huber |
| 二分类 | Binary Crossentropy |
| 多分类 | Categorical Crossentropy |

#### Python实现
```python
def mse_loss(y_true, y_pred):
    """均方误差"""
    return np.mean((y_true - y_pred) ** 2)

def mae_loss(y_true, y_pred):
    """平均绝对误差"""
    return np.mean(np.abs(y_true - y_pred))

def binary_crossentropy(y_true, y_pred):
    """二元交叉熵"""
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(y_true * np.log(y_pred) + (1 - y_true) * np.log(1 - y_pred))

def categorical_crossentropy(y_true, y_pred):
    """多分类交叉熵"""
    epsilon = 1e-15
    y_pred = np.clip(y_pred, epsilon, 1 - epsilon)
    return -np.mean(np.sum(y_true * np.log(y_pred), axis=1))
```

## 第三周：反向传播与优化器

### 3.1 反向传播

#### 什么是反向传播？

**直观理解**：反向传播就是"从错误中学习"，从输出层向输入层传播误差，调整权重。

```
前向传播：输入 → 隐藏 → 输出（计算预测值）
反向传播：输出 → 隐藏 → 输入（计算梯度，更新权重）

类比：
考试：答题 → 得分
学习：看错题 → 找原因 → 改进
```

#### 核心思想：链式法则

```
损失L对权重w的梯度 = ∂L/∂w

链式法则：
∂L/∂w = ∂L/∂y × ∂y/∂z × ∂z/∂w

一层一层往回传
```

#### 反向传播流程

```
1. 前向传播：计算每一层的输出
2. 计算损失：比较预测和真实值
3. 反向传播：计算每一层的梯度
4. 更新权重：w = w - lr × 梯度
```

### 3.2 优化器

#### 什么是优化器？

**直观理解**：优化器决定如何更新权重，让损失下降得更快更好。

```
梯度下降：沿坡往下走
优化器：决定怎么走（步子大小、方向修正等）
```

#### 常用优化器

**SGD（随机梯度下降）**：
```
w = w - lr × 梯度

最简单，但可能震荡
```

**SGD + Momentum（动量）**：
```
v = β × v + 梯度
w = w - lr × v

像小球滚动，有惯性，减少震荡
```

**Adam（最常用）**：
```
结合动量和自适应学习率
自动调整每个参数的学习率

大多数情况首选Adam
```

#### 优化器选择指南

| 场景 | 推荐优化器 |
|------|------------|
| 通用 | Adam |
| 追求最好泛化 | SGD + Momentum |
| 稀疏数据 | Adam |
| 快速实验 | Adam |

#### Python实现
```python
class SGD:
    def __init__(self, learning_rate=0.01):
        self.lr = learning_rate
    
    def update(self, params, grads):
        return params - self.lr * grads

class SGDMomentum:
    def __init__(self, learning_rate=0.01, momentum=0.9):
        self.lr = learning_rate
        self.momentum = momentum
        self.velocity = None
    
    def update(self, params, grads):
        if self.velocity is None:
            self.velocity = np.zeros_like(params)
        self.velocity = self.momentum * self.velocity + grads
        return params - self.lr * self.velocity

class Adam:
    def __init__(self, learning_rate=0.001, beta1=0.9, beta2=0.999):
        self.lr = learning_rate
        self.beta1 = beta1
        self.beta2 = beta2
        self.m = None  # 一阶矩
        self.v = None  # 二阶矩
        self.t = 0     # 时间步
    
    def update(self, params, grads):
        self.t += 1
        if self.m is None:
            self.m = np.zeros_like(params)
            self.v = np.zeros_like(params)
        
        self.m = self.beta1 * self.m + (1 - self.beta1) * grads
        self.v = self.beta2 * self.v + (1 - self.beta2) * grads**2
        
        m_hat = self.m / (1 - self.beta1**self.t)
        v_hat = self.v / (1 - self.beta2**self.t)
        
        return params - self.lr * m_hat / (np.sqrt(v_hat) + 1e-8)
```

## 第四周：正则化技术

### 4.1 什么是正则化？

**直观理解**：正则化就是防止模型"死记硬背"，让它学会"举一反三"。

```
没有正则化：训练99%，测试70%（过拟合）
有正则化：训练90%，测试85%（泛化好）
```

### 4.2 Dropout

#### 什么是Dropout？

**直观理解**：训练时随机"关掉"一些神经元，防止依赖。

```
训练时：        测试时：
○ ○ × ○ ○      ○ ○ ○ ○ ○
○ × ○ ○ ×  →   ○ ○ ○ ○ ○
× ○ ○ ○ ○      ○ ○ ○ ○ ○

× 表示被关掉的神经元
```

**类比**：
- 团队合作时，随机有人请假
- 迫使每个人都能独立完成任务
- 最终团队更强大

#### Python实现
```python
class Dropout:
    def __init__(self, rate=0.5):
        self.rate = rate  # 关掉的比例
    
    def forward(self, x, training=True):
        if training:
            # 随机关掉
            mask = np.random.binomial(1, 1 - self.rate, size=x.shape)
            return x * mask / (1 - self.rate)  # 缩放
        else:
            return x  # 测试时全部使用
```

### 4.3 Batch Normalization

#### 什么是Batch Normalization？

**直观理解**：让每一层的输入都保持在"正常范围"，加速训练。

```
没有BN：输入分布不断变化，训练困难
有BN：输入分布稳定，训练更快
```

**类比**：
- 没有BN：每次考试难度不同，学生适应困难
- 有BN：每次考试难度标准化，学生稳定发挥

#### Python实现
```python
class BatchNorm:
    def __init__(self, n_features, momentum=0.9):
        self.gamma = np.ones(n_features)
        self.beta = np.zeros(n_features)
        self.momentum = momentum
        self.running_mean = np.zeros(n_features)
        self.running_var = np.ones(n_features)
    
    def forward(self, x, training=True):
        if training:
            mean = x.mean(axis=0)
            var = x.var(axis=0)
            # 更新running统计量
            self.running_mean = self.momentum * self.running_mean + (1 - self.momentum) * mean
            self.running_var = self.momentum * self.running_var + (1 - self.momentum) * var
        else:
            mean = self.running_mean
            var = self.running_var
        
        # 标准化
        x_norm = (x - mean) / np.sqrt(var + 1e-8)
        # 缩放和偏移
        return self.gamma * x_norm + self.beta
```

### 4.4 正则化方法总结

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| Dropout | 随机关掉神经元 | 全连接层 |
| BatchNorm | 标准化每层输入 | 加速训练 |
| L1正则化 | 惩罚权重绝对值 | 特征选择 |
| L2正则化 | 惩罚权重平方 | 防止过拟合 |
| 数据增强 | 增加训练数据 | 图像任务 |

## PyTorch实现

### 完整神经网络示例

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader, TensorDataset

class SimpleNN(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(SimpleNN, self).__init__()
        self.fc1 = nn.Linear(input_size, hidden_size)
        self.relu = nn.ReLU()
        self.dropout = nn.Dropout(0.5)
        self.bn = nn.BatchNorm1d(hidden_size)
        self.fc2 = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        x = self.fc1(x)
        x = self.bn(x)
        x = self.relu(x)
        x = self.dropout(x)
        x = self.fc2(x)
        return x

# 创建数据
X = torch.randn(1000, 10)
y = torch.randint(0, 2, (1000,))

# 创建数据加载器
dataset = TensorDataset(X, y)
dataloader = DataLoader(dataset, batch_size=32, shuffle=True)

# 创建模型
model = SimpleNN(10, 64, 2)
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 训练循环
for epoch in range(100):
    model.train()
    total_loss = 0
    
    for batch_X, batch_y in dataloader:
        optimizer.zero_grad()
        outputs = model(batch_X)
        loss = criterion(outputs, batch_y)
        loss.backward()
        optimizer.step()
        total_loss += loss.item()
    
    if (epoch + 1) % 10 == 0:
        print(f"Epoch {epoch+1}, Loss: {total_loss/len(dataloader):.4f}")
```
